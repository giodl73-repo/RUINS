# RUINS Review Organization

RUINS reviews protect the boundary between surviving evidence, responsible
inference, legend, reconstruction, and modern projection. The panel asks whether
an artifact explains absence without filling it too quickly.

## Panel

| ID | Role | Lens |
|---|---|---|
| [R-1](panel/R-1-fragment-custodian.md) | Fragment Custodian | Surviving traces, material evidence, source condition |
| [R-2](panel/R-2-absence-historian.md) | Absence Historian | Loss events, institutional collapse, archival gaps |
| [R-3](panel/R-3-reconstruction-skeptic.md) | Reconstruction Skeptic | Claim grading, contested reconstructions, speculative inflation |
| [R-4](panel/R-4-cultural-memory-reader.md) | Cultural Memory Reader | Later myth, identity, memory, imitation, and symbolic afterlife |
| [R-5](panel/R-5-sibling-boundary-editor.md) | Sibling Boundary Editor | Hand-offs to MAXIM, LUCIA, MUNDUS, FONTES, PORTO, CANON, RITE, GENES, and CERES |

## Editorial

| ID | Role | Lens |
|---|---|---|
| E-1 | Paper Editor | Keeps papers informational, cited, and scoped |
| E-2 | Safety Editor | Blocks looting-enabling, sensational, or unsupported claims |

## Review Order

1. Use the full panel for method papers, seed-corpus selection, and any proposed
   taxonomy.
2. Use the Reconstruction Skeptic before promoting a claim from legend or
   hypothesis to probable reconstruction.
3. Use the Safety Editor before publishing site-specific details that could
   enable damage, looting, or misrepresentation.
4. Use the Sibling Boundary Editor before expanding RUINS into place geography,
   source custody, ritual systems, family continuity, or production economics.

## Finding Severity

| Severity | Meaning |
|---|---|
| P1 | Blocks publication because the artifact confuses evidence with speculation, violates safety boundaries, or misplaces ownership. |
| P2 | Should fix before relying on the artifact in a paper or downstream system. |
| P3 | Useful refinement that can wait. |
