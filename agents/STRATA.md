# STRATA

## Assignment

STRATA studies time, layers, and loss processes. It keeps RUINS from collapsing
long abandonment, repeated damage, reuse, excavation, restoration, and later
memory into one dramatic event.

## Responsibilities

- Build event chronologies for each case.
- Separate sudden destruction from slow loss.
- Track institutional, environmental, political, material, and archival
  processes.
- Identify when "collapse" is too simple for the evidence.
