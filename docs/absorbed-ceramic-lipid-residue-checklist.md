# Absorbed Ceramic Lipid Residue Checklist

Use this checklist when cuisine, dairying, plant processing, vessel use,
pastoralism, trade, diet, climate, or foodway history is reconstructed from
absorbed pottery residues, fatty acids, alkanoic acids, GC-MS, GC-C-IRMS, stable
carbon isotopes, lipid biomarkers, sherd fabrics, visible residues, or reference
fats.

## Required Separations

| Surface | Required custody question |
|---|---|
| Vessel | Which vessel, sherd, fabric, form, context, phase, and use surface were sampled? |
| Residue location | Is the residue visible surface deposit, absorbed wall lipid, charred crust, soil contaminant, or conservation residue? |
| Preservation | What survival, degradation, leaching, heating, reuse, burial, and cleaning controls are locked? |
| Extraction | Which extraction, derivatization, instrument, blank, replicate, and threshold produced the signal? |
| Biomarker | Is the assignment fatty acid class, dairy fat, adipose fat, plant wax, millet marker, beeswax, resin, or mixed? |
| Isotope | Which reference range, local ecology, animal diet, C3/C4 baseline, and uncertainty support the call? |
| Commodity | Is the claim milk, meat, plant, mixed food, storage, cooking, repair, waterproofing, or unknown use? |
| Scale | Is the inference vessel event, repeated vessel use, site practice, herd economy, cuisine, or regional subsistence? |
| Sibling | Are animal, plant, ceramic, place, trade, and foodway syntheses handed to the right systems? |
| Memory | Are first-dairying, ancient-menu, pastoralist, and recipe stories kept below source-locked inference? |

## Refuse Unless Separately Locked

- Lipid residue is an exact meal.
- Dairy fat is direct individual diet.
- Vessel residue is a complete recipe.
- Milk processing is whole herd economy.
- Pottery residue alone proves domestication.
- Isotope values are an ethnic identity or migration route.
- A first-dairying headline is complete foodway history.
