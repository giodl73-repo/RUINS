# Ancient Pathogen Genome Checklist

Use this checklist when disease presence, outbreak, epidemic, pandemic,
mortality event, virulence, transmission route, vector, host reservoir,
geographic origin, spread path, strain continuity, population impact, immune
selection, historical event attribution, or public plague memory is
reconstructed from ancient pathogen DNA, dental pulp, petrous bone, skeletal
remains, metagenomic reads, capture enrichment, shotgun sequencing, pathogen
genome coverage, SNP calls, virulence genes, plasmids, phylogenies, molecular
clocks, screening positives, negative samples, or historical burial contexts.

## Required Separations

| Surface | Required custody question |
|---|---|
| Sample | Which tooth, pulp, bone, calculus, burial, individual, cemetery, and lab extract is analyzed? |
| Context | Is the burial individual, multiple, mass, disturbed, curated, dated, or historically associated? |
| Screening | Is the signal shotgun hit, capture positive, qPCR, metagenomic profile, or assembled genome? |
| Authentication | Are damage, fragment length, contamination, blanks, mapping, and independent replication recorded? |
| Coverage | Is the result low-coverage hit, partial genome, high-coverage genome, SNP panel, or absent signal? |
| Taxon | Is the assignment species, strain, lineage, plasmid, virulence marker, or ambiguous pathogen call? |
| Chronology | Is the date direct, associated, modelled, molecular-clock, or historically inferred? |
| Epidemiology | Are infection, disease, cause of death, outbreak, vector, reservoir, and transmission separated? |
| Scale | Are individual, cemetery, region, migration, pandemic, and population-impact claims separated? |
| Memory | Are plague origin, pandemic certainty, collapse, blame, and public health analogy contained? |

## Refuse Unless Separately Locked

- Pathogen read is disease diagnosis.
- Positive genome is cause of death.
- Mass burial is proven outbreak.
- Virulence gene is transmission route.
- Plasmid presence is vector certainty.
- Phylogenetic branch is migration route.
- Molecular clock is exact origin.
- Plague headline is population collapse.
