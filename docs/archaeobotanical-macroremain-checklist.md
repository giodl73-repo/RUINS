# Archaeobotanical Macroremain Checklist

Use this checklist when diet, crop choice, domestication, crop-processing stage,
storage, harvesting season, field ecology, weed ecology, fuel use, animal
fodder, room function, household activity, market supply, agricultural economy,
ritual deposit, famine, migration, or public first-farming memory is
reconstructed from charred seeds, cereal grains, chaff, rachis segments,
spikelet forks, weed seeds, fruits, nutshells, pulses, flotation samples, heavy
fractions, waterlogged remains, desiccated remains, mineralized remains, seed
counts, ubiquity, density, crop ratios, immature grains, or reference
collections.

## Required Separations

| Surface | Required custody question |
|---|---|
| Sample | Which context, volume, fraction, mesh size, flot, heavy residue, and collection record is analyzed? |
| Preservation | Is survival charred, waterlogged, desiccated, mineralized, contaminated, or redeposited? |
| Recovery | Were sampling strategy, flotation method, screen size, and lab sorting recorded? |
| Taxon | Is the identification family, genus, species, crop type, wild taxon, weed, or unknown? |
| Taphonomy | Was burning, dumping, storage loss, hearth rake-out, dung fuel, bioturbation, or intrusion tested? |
| Quantification | Are count, weight, density, ubiquity, ratio, absence, and sample volume separated? |
| Processing | Are harvest, threshing, winnowing, sieving, dehusking, storage, cooking, and discard separated? |
| Spatial pattern | Are household room, pit, hearth, silo, field, dump, and settlement-scale inferences separated? |
| Synthesis | Are diet, domestication, economy, market, migration, famine, and social claims handed to siblings? |
| Memory | Are first-farming, ancient meal, granary, famine, and ritual deposit stories contained? |

## Refuse Unless Separately Locked

- Charred seed is exact crop inventory.
- Flotation count is complete diet.
- Chaff ratio is final crop-processing stage.
- Weed seed is field ecology.
- Storage pit seed cache is full storage system.
- Immature grain ratio is harvest calendar.
- Seed absence is crop absence.
- First-farming headline is domestication proof.
