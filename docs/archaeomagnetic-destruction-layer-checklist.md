# Archaeomagnetic Destruction-Layer Checklist

Use this checklist when dating, destruction event, military campaign, conquest,
fire episode, biblical/historical narrative, geomagnetic anomaly, settlement
chronology, or regional synchronization is reconstructed from fired mudbrick,
burnt floors, hearths, kilns, ceramics, slag, archaeomagnetic direction,
archaeointensity, thermoremanent magnetization, secular variation curves, or
paleomagnetic models.

## Required Separations

| Surface | Required custody question |
|---|---|
| Feature | Which floor, mudbrick, hearth, kiln, ceramic, slag, or destruction layer was sampled? |
| Heating | Was the material heated enough, in place, and in a single event to lock thermoremanent magnetization? |
| Position | Did the sample cool in its measured orientation, or was it collapsed, tilted, reused, or disturbed? |
| Orientation | How were azimuth, dip, bedding, block orientation, and sampling uncertainty recorded? |
| Mineralogy | Which magnetic minerals, alteration, anisotropy, domain state, and stability checks support the signal? |
| Measurement | Which direction, intensity, demagnetization, correction, lab, and reproducibility controls are locked? |
| Curve | Which regional secular variation or archaeointensity curve and uncertainty envelope is used? |
| Anchor | Is the reference date independent text, inscription, radiocarbon, typology, or historical synthesis? |
| Event | Is the claim fire episode, destruction layer, campaign, conquest, citywide event, or regional map? |
| Memory | Are biblical-proof, campaign-map, exact-date, and radiocarbon-replacement stories contained? |

## Refuse Unless Separately Locked

- Archaeomagnetism is an exact destruction date.
- Magnetic similarity is certain campaign attribution.
- A destruction layer directly confirms a text.
- One fired feature is a complete site chronology.
- Magnetic dating replaces radiocarbon or historical custody.
- Multiple matched layers are a complete conquest map.
- Public biblical-proof memory is archaeology.
