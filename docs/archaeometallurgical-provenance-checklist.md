# Archaeometallurgical Provenance Checklist

Use this checklist when ore source, mine connection, smelting location,
workshop, metal circulation, trade route, technological recipe, furnace
efficiency, production scale, recycling, alloying, political economy, craft
identity, or public ancient-industry memory is reconstructed from slag, slag
inclusions, metal artefacts, ores, lead isotopes, trace elements, XRF, pXRF,
LA-ICP-MS, SEM-EDS, XRD, metallography, corrosion products, furnace remains,
tuyere fragments, crucibles, prills, or compositional databases.

## Required Separations

| Surface | Required custody question |
|---|---|
| Sample | Which artefact, slag, inclusion, ore, furnace, crucible, tuyere, prill, or corrosion layer was sampled? |
| Context | Is the sample production waste, finished object, repair, recycling residue, workshop debris, or displaced find? |
| Method | Which analytical method, calibration, detection limit, and preparation protocol produced the signature? |
| Alteration | Are corrosion, contamination, burial chemistry, conservation treatment, and sampling damage separated? |
| Ore field | Which ore source candidates, geological overlaps, and database limits are explicit? |
| Mixing | Are alloying, recycling, remelting, scrap, and multi-source blending gated before provenance claims? |
| Production stage | Is the evidence ore extraction, smelting, refining, smithing, casting, repair, use, or discard? |
| Probability | Is the match exact, probable, compatible, overlapping, ambiguous, or refused? |
| Exchange | Are mine source, workshop, route, trade, market, polity, and identity separately owned? |
| Memory | Are ancient industry, trade network, technological superiority, and origin stories contained? |

## Refuse Unless Separately Locked

- Lead isotope ratio is exact mine.
- Slag chemistry is complete smelting recipe.
- Slag inclusion is direct workshop.
- XRF cluster is trade route.
- Ore database match is solved provenance.
- Furnace residue is production scale.
- Alloy composition is ethnic or craft identity.
- Metal circulation map is political economy.
