# Astronomical Alignment Reconstruction Checklist

Use this checklist when ruins are interpreted through solar, lunar, stellar,
horizon, light-beam, shadow, passage, slab, petroglyph, roof-box, or
architectural orientation evidence.

## Required Separations

| Surface | Required custody question |
|---|---|
| Fabric | What survives, moved, eroded, restored, reconstructed, or is inaccessible? |
| Observation | What date, time, cycle, weather, horizon, and viewpoint are involved? |
| Model | Is the claim observed directly, simulated, digitally restored, or archival? |
| Marking | What light, shadow, beam, petroglyph, passage, aperture, or alignment is recorded? |
| Calendar | Is the claim an event, window, marker, cycle, or full system? |
| Meaning | Are ritual, political, agricultural, and social functions separately sourced? |
| Memory | Is archaeoastronomy public memory separated from ancient practice? |

## Refuse Unless Separately Locked

- Alignment is design certainty.
- A light or shadow event is a complete calendar system.
- A model is direct ancient observation.
- Astronomical function proves ritual meaning.
- Public archaeoastronomy memory is unbroken tradition.
