# Biometric Trace Reconstruction Checklist

Use this checklist when age, sex, identity, craft role, apprenticeship, labor
organization, or social division is reconstructed from fingerprints, palmprints,
footprints, ridge breadth, ridge density, impressions, scans, or biometric
models.

## Required Separations

| Surface | Required custody question |
|---|---|
| Trace | What impression, ridge field, print, scan, or partial mark is being measured? |
| Substrate | What clay, ceramic, plaster, sediment, or other material received the trace? |
| Formation | Was the mark made during manufacture, handling, use, post-deposition, or recovery? |
| Preservation | What smoothing, burnishing, breakage, erosion, conservation, or selection bias affected survival? |
| Distortion | How did shrinkage, firing, pressure, moisture, angle, or scan processing alter dimensions? |
| Measurement | Which ridge-breadth, ridge-density, image, or model workflow produced the metric? |
| Reference | Which modern or comparative population supports the biometric category? |
| Inference | Is the output age range, sex category, broad demographic tendency, or individual claim? |
| Behavior | Are craft role, learning, labor organization, gender system, and economy separately sourced? |
| Memory | Are public maker, child, artisan, or gender-work stories kept separate from the trace? |

## Refuse Unless Separately Locked

- A fingerprint is an exact named maker.
- Ridge density is certain sex.
- Ridge breadth is precise age.
- A measurable trace is a complete apprenticeship history.
- A sherd sample is a workforce census.
- A demographic tendency is a whole gender system.
- A public maker story is direct social history.
