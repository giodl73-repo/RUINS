# Canopy LiDAR Settlement Checklist

Use this checklist when hidden settlement, urban form, roads, causeways,
terraces, field systems, reservoirs, canals, population, political integration,
land-use intensity, abandonment, conservation priority, or public lost-city
memory is reconstructed from airborne LiDAR, point clouds, ground returns,
digital terrain models, canopy filtering, hillshade, slope, sky-view factor,
feature extraction, machine learning, survey polygons, environmental LiDAR, or
remote-sensing archives.

## Required Separations

| Surface | Required custody question |
|---|---|
| Sensor flight | Which flight, instrument, date, altitude, swath, and coverage polygon produced the data? |
| Point cloud | What point density, return classes, noise, and classification workflow are available? |
| Canopy filtering | Which vegetation-removal and ground-classification choices shaped the terrain model? |
| Terrain model | Is the claim based on raw points, DTM, DEM, hillshade, slope, sky-view factor, or composite view? |
| Feature candidate | Is the feature a candidate mound, platform, terrace, road, canal, reservoir, or natural landform? |
| Ground truth | Which candidates were checked by pedestrian survey, excavation, surface collection, or legacy map? |
| Chronology | Does LiDAR show form only, or is dating independently locked? |
| Function | Are settlement, agricultural, hydraulic, ritual, defensive, and natural interpretations separated? |
| Scale | Are survey polygon, extrapolation, omission/commission error, and regional model separated? |
| Population | Are structure count, occupancy assumption, household size, period, and uncertainty explicit? |
| Memory | Are lost city, rediscovered civilization, collapse, and conservation-complete map stories contained? |

## Refuse Unless Separately Locked

- Point cloud is complete settlement inventory.
- Hillshade is excavation-equivalent confirmation.
- Feature candidate is confirmed building.
- Road trace is complete political network.
- Structure count is exact population.
- Survey polygon is whole-region map.
- Environmental scan is archaeological coverage.
- Lost-city image is rediscovered civilization.
