## Problem

Absorbed lipid residues in pottery can preserve durable food and commodity
signals, but a sherd is not a meal record. Fatty acids, stable carbon isotope
values, lipid biomarkers, visible residues, sherd fabric, vessel form, reference
fat ranges, and public first-dairying stories can be collapsed into exact
recipes, direct individual diet, complete herd economy, domestication proof,
ethnic identity, trade route, climate proxy, or whole cuisine.

## Core Rule

Ceramic lipid claims must keep vessel, sherd, residue location, extraction,
preservation, contamination, biomarker assignment, isotopic reference range,
commodity class, vessel-use inference, subsistence scale, sibling synthesis, and
public food memory separate.
