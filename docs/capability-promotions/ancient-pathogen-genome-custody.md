# Ancient Pathogen Genome Custody

**Capability id:** `ruins-capability:ancient-pathogen-genome-custody`
**Promotion:** Candidate -> Active -> Core
**Trigger paper:** `research/publications/panel-ancient-pathogen-genome-custody/`
**Rubric:** v1.63

## Problem

Ancient pathogen genomics can identify microbial DNA in archaeological remains
and place strains in evolutionary history. It is also prone to overpromotion.
Ancient pathogen DNA, dental-pulp reads, skeletal metagenomic hits,
capture-enriched sequences, pathogen genomes, coverage values, SNP calls,
virulence genes, plasmid presence, phylogenetic branches, molecular-clock dates,
screening positives, negative samples, mass-burial contexts, and public plague
headlines can become complete disease diagnoses, certain causes of death, proven
outbreaks, pandemic origins, exact transmission routes, vector certainty, host
reservoirs, population collapse, migration proof, immune-selection histories,
historical event attributions, or solved plague stories.

## Core Rule

Pathogen genome claims must keep sample, skeletal context, screening method,
authentication, contamination, genome coverage, taxon assignment, strain
placement, virulence marker, host relation, chronology, epidemiological scale,
historical synthesis, and public disease memory separate.

## Required Controls

- Pathogen-genome gate.
- Source lock.
- Page-anchor audit.
- Refusal register.
- Cultural-memory register.
- Sibling-boundary register.
- Cross-case checklist.

## Decision

Promote directly to Core under rubric v1.63. Ancient pathogen genome custody
creates a reusable control for plague, ancient infectious disease, virulence,
transmission, outbreak, mortality, and pandemic-history claims.
