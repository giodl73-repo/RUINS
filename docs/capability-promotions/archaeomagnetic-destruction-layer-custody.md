## Problem

Archaeomagnetic dating can turn burned archaeological materials into precise
chronological anchors, but a magnetic signal is not a witnessed event. Fired
mudbrick, burnt floors, hearths, kilns, ceramics, slag, directions,
intensities, secular variation curves, reference dates, and public biblical
proof stories can be collapsed into exact destruction dates, campaign
attributions, textual confirmation, regional conquest maps, or replacement of
radiocarbon and historical custody.

## Core Rule

Archaeomagnetic claims must keep feature, heating event, cooling position,
disturbance, sample orientation, mineralogy, lab measurement, calibration curve,
reference date, event correlation, campaign synthesis, and public proof memory
separate.
