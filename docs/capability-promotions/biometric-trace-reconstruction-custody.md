# Biometric Trace Reconstruction Custody

**Capability id:** `ruins-capability:biometric-trace-reconstruction-custody`
**Promotion:** Candidate -> Active -> Core
**Trigger paper:** `research/publications/panel-ancestral-pueblo-fingerprint-custody/`
**Rubric:** v1.38

## Problem

Ancient fingerprints and related biometric traces can preserve unusually direct
human contact, but that contact is easily overread. Ridge breadth, ridge
density, ceramic shrinkage, reference-population choice, and statistical output
can be collapsed into exact identity, certain sex, precise age, apprenticeship,
or total labor organization.

## Core Rule

Biometric traces must keep trace, substrate, formation, preservation,
distortion, measurement, reference population, model uncertainty, demographic
inference, behavior, and public memory separate.

## Required Controls

- Biometric trace gate.
- Source lock.
- Page-anchor audit.
- Refusal register.
- Cultural-memory register.
- Sibling-boundary register.
- Cross-case checklist.

## Decision

Promote directly to Core under rubric v1.38. The Ancestral Pueblo fingerprint
case creates a control that is general enough for fingerprints, palmprints,
footprints, and other biometric impressions, and the rubric cap prevents trace
metrics from becoming identity, sex, age, biography, apprenticeship, workforce,
or gender-system claims.
