# Canopy LiDAR Settlement Custody

**Capability id:** `ruins-capability:canopy-lidar-settlement-custody`
**Promotion:** Candidate -> Active -> Core
**Trigger paper:** `research/publications/panel-canopy-lidar-settlement-custody/`
**Rubric:** v1.59

## Problem

Airborne LiDAR can expose settlement traces beneath vegetation, including
platforms, roads, causeways, terraces, fields, reservoirs, canals, and broader
landscape organization. Those traces are often promoted too fast. Point clouds,
ground returns, digital terrain models, hillshades, slope views, sky-view
factors, feature candidates, survey polygons, structure counts, environmental
scans, and public lost-city images can be collapsed into complete settlement
inventories, exact populations, political organization, urban plans, road
networks, chronologies, excavation-equivalent confirmation, environmental
determinism, rediscovered civilizations, conservation-complete maps, or solved
collapse stories.

## Core Rule

LiDAR settlement claims must keep sensor flight, point cloud, return
classification, canopy filtering, terrain model, visualization, feature
candidate, ground-truth sample, chronology, functional interpretation,
population model, landscape boundary, publication image, and public discovery
memory separate.

## Required Controls

- LiDAR-settlement gate.
- Source lock.
- Page-anchor audit.
- Refusal register.
- Cultural-memory register.
- Sibling-boundary register.
- Cross-case checklist.

## Decision

Promote directly to Core under rubric v1.59. Canopy LiDAR settlement custody
creates a reusable control for hidden-site discovery, feature extraction,
population, urbanism, road-network, conservation, and lost-city claims.
