# Ceramic Petrography Fabric Custody

**Capability id:** `ruins-capability:ceramic-petrography-fabric-custody`
**Promotion:** Candidate -> Active -> Core
**Trigger pressure:** ceramic thin-section petrography, combined petrography and
geochemistry, and automated fabric classification
**Rubric:** v1.68

## Problem

Ceramic petrography can connect archaeological ceramics to production choices,
raw materials, and broad provenance zones. It can also overpromote fabric
similarity. Thin sections, clay matrices, mineral inclusions, tempers, voids,
grain size, optical mineralogy, firing textures, fabric groups, SEM/XRD, NAA,
pXRF/XRF, geochemical clusters, raw clay comparisons, style overlap, image
classification, automated fabric models, and public trade-route stories can
collapse into exact workshop, direct route, certain clay source, complete
recipe, firing method, production system, ethnicity, migration, political
boundary, market map, or complete interaction sphere.

## Core Rule

Ceramic petrography claims must keep sherd/object, context, sampling design,
thin section, fabric description, grouping, raw material comparison, analytical
pairing, production inference, provenance inference, and public memory separate.

## Required Controls

- Ceramic petrography fabric gate.
- Source lock.
- Page-anchor audit.
- Refusal register.
- Cultural-memory register.
- Sibling-boundary register.
- Cross-case checklist.

## Source Pressure

| Source family | Pressure exposed |
|---|---|
| Ceramic petrography method papers | Thin-section description, fabric grouping, matrix/inclusion interpretation, and provenance/technology claims require explicit sample and interpretation custody. |
| Combined petrography and geochemistry | Bulk chemical groups, petrographic fabrics, raw clay samples, and geological provenance do not always align cleanly. |
| Automated fabric classification | Model labels and image classes need training-data, expert-label, and model-scope custody before they support archaeological inference. |

## Decision

Promote directly to Core under rubric v1.68. Ceramic petrography fabric custody
fills the Material Trace and Production Systems gap for ceramic fabrics by
separating thin-section evidence, compositional grouping, model classification,
production inference, provenance inference, and public exchange memory.
