# Dendrochronological Timber-Date Custody

**Capability id:** `ruins-capability:dendrochronological-timber-date-custody`
**Promotion:** Candidate -> Active -> Core
**Trigger paper:** `research/publications/panel-mesa-verde-dendrochronology-timber-date-custody/`
**Rubric:** v1.53

## Problem

Dendrochronology can assign annual dates to wood and often gives archaeology
its sharpest chronology. That precision can be overpromoted. Tree-ring dates,
terminal rings, felling clusters, reference chronology matches, timber
provenance, repair episodes, reused beams, drought rings, and beam counts can be
collapsed into exact building completion, occupation span, abandonment date,
drought-causation proof, community size, timber-trade maps, forest-management
systems, household biographies, or whole settlement histories.

## Core Rule

Tree-ring claims must keep sample, tree, ring sequence, terminal ring, felling
event, reuse, construction context, repair/remodeling, chronology match,
provenance, climate signal, occupation inference, and public exact-year memory
separate.

## Required Controls

- Timber-date gate.

## Decision

Promote directly to Core under rubric v1.53. Dendrochronological
timber-date custody creates a reusable control for annual dating, construction
sequence, reuse, provenance, drought, and settlement-history claims.
