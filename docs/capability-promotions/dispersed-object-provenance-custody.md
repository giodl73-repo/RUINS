## Overclaim Pressure

A materially surviving object can be mistaken for a context-complete object.
Museum possession, digital cataloguing, legal transfer, return, loan, display,
and public restitution memory can be collapsed into one custody claim.

| Artifact | Path | What it controls |

| Refusal register | `research/publications/panel-benin-bronzes-dispersed-custody/refusal-register.md` | Blocks same-provenance, all-bronze, digital-equals-return, return-equals-repair, and legal-title overclaims. |

## ASHLAR Decision

Record `dispersed-object-provenance-custody` as Active. It is ready for reuse
when RUINS handles looted, dispersed, or partially returned object corpora, but
it should become Core only after a second unlike case proves that the controls
generalize beyond Benin.
