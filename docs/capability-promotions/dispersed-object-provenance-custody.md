## Overclaim Pressure

A materially surviving object can be mistaken for a context-complete object.
Museum possession, digital cataloguing, legal transfer, return, loan, display,
and public restitution memory can be collapsed into one custody claim.

## ASHLAR Decision

Record `dispersed-object-provenance-custody` as Core. Benin Bronzes and
Parthenon Sculptures show that the same controls work across different
dispersed-object conditions: colonial looting and global restitution, and
contested removal/title/display/reunification. The Core method separates object
identity, provenance, holding, legal/governance action, physical movement,
digital/replica display, and living/public memory.
