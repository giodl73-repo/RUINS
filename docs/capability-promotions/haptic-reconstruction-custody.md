# Haptic Reconstruction Custody

**Capability id:** `ruins-capability:haptic-reconstruction-custody`
**Promotion:** Candidate -> Active -> Core
**Trigger pressure:** haptic digital archaeology, haptic augmented reality, and
3D-printed tactile heritage replicas
**Rubric:** v1.66

## Problem

Touch access is powerful in heritage work because many archaeological and museum
objects cannot be handled directly. Haptic feedback, tactile replicas,
3D-printed models, touch tables, gloves, styluses, vibration, force feedback,
haptic augmented reality, virtual handling, accessibility replicas, user-study
enthusiasm, and "touching the past" stories can collapse mediated access into
direct contact with the original object. They can also become claims about
original material, exact surface feel, weight, resistance, ancient use
experience, craft action, ritual meaning, solved conservation, or final
interpretation.

## Core Rule

Haptic reconstruction claims must keep original object, conservation status,
capture method, geometry, surface texture, material and weight, haptic device,
alignment, user study, access claim, and public memory separate.

## Required Controls

- Haptic reconstruction gate.
- Source lock.

- Cross-case checklist.

## Source Pressure

| Source family | Pressure exposed |
|---|---|
| Haptics in digital archaeology | Haptic feedback can reveal object features and improve exploration, but similarity ratings and expert/novice differences must not become object equivalence. |
| Haptic augmented reality | Touch-like museum interaction creates an illusion of handling an untouchable object and needs clear original/interface separation. |
| 3D-printed tactile replicas | Printed or fabricated access models improve inclusion while changing material, weight, scale, surface, and conservation status. |

## Decision

Promote directly to Core under rubric v1.66. Haptic reconstruction custody fills
the Reconstruction, Replica, and Sensory Access family gap by separating
touch-like access from original contact, object-equivalent fabric, complete
sensory recovery, and ancient embodied experience.
