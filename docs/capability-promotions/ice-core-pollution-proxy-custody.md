# Ice-Core Pollution Proxy Custody

**Capability id:** `ruins-capability:ice-core-pollution-proxy-custody`
**Promotion:** Candidate -> Active -> Core
**Trigger paper:** `research/publications/panel-greenland-ice-core-lead-proxy-custody/`
**Rubric:** v1.47

## Problem

Ice cores can preserve high-resolution pollution records that illuminate ancient
mining, smelting, and industrial activity, but an ice-core proxy is not a direct
ledger. Lead concentrations, isotope ratios, annual layers, deposition flux,
transport models, source apportionment, and historical-event alignments can be
collapsed into exact economic output, single-event proof, imperial expansion,
plague or war causation, population exposure, IQ loss, civilizational decline,
or complete industrial history.

## Core Rule

Ice-core pollution claims must keep core, chronology, contaminant, deposition
process, source region, transport model, proxy-to-output conversion, event
correlation, exposure model, scale, uncertainty, and public decline memory
separate.

## Required Controls

- Ice-core proxy gate.
- Source lock.
- Page-anchor audit.
- Refusal register.
- Cultural-memory register.
- Sibling-boundary register.
- Cross-case checklist.

## Decision

Promote directly to Core under rubric v1.47. The Greenland/Arctic lead-pollution
case creates a reusable control for environmental proxies used as historical,
economic, and public-health evidence.
