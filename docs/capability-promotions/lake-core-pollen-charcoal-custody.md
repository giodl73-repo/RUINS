# Lake-Core Pollen Charcoal Custody

**Capability id:** `ruins-capability:lake-core-pollen-charcoal-custody`
**Promotion:** Candidate -> Active -> Core
**Trigger paper:** `research/publications/panel-rapa-nui-lake-core-collapse-custody/`
**Rubric:** v1.50

## Problem

Lake-core pollen and charcoal can reveal vegetation and fire history, but a core
is not a social collapse ledger. Pollen decline, grass expansion, charcoal
particles, sediment inwash, diatoms, geochemistry, isotope values, and public
ecocide stories can be collapsed into exact deforestation dates, whole-island
maps, direct human overuse proof, population collapse, warfare, famine, climate
determinism, or moral parables.

## Core Rule

Lake-core proxy claims must keep core, catchment, proxy type, preservation,
transport, chronology, taxon assignment, fire signal, human/climate attribution,
spatial scale, social inference, and public collapse memory separate.

## Required Controls

- Lake-core proxy gate.
- Source lock.
- Page-anchor audit.
- Refusal register.
- Cultural-memory register.
- Sibling-boundary register.
- Cross-case checklist.

## Decision

Promote directly to Core under rubric v1.50. The Rapa Nui case creates a
reusable control for pollen/charcoal records, deforestation narratives, and
collapse or resilience public memory.
