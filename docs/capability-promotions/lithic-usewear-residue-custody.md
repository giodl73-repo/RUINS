# Lithic Use-Wear Residue Custody

**Capability id:** `ruins-capability:lithic-usewear-residue-custody`
**Promotion:** Candidate -> Active -> Core
**Trigger paper:** `research/publications/panel-stone-tool-usewear-residue-custody/`
**Rubric:** v1.43

## Problem

Stone-tool use-wear and residues can support tool-function and worked-material
claims, but traces are ambiguous, surface texture is heterogeneous, residues can
transfer or degrade, and experimental analogues are bounded. Without custody,
edge polish or a residue image can become exact function, hunting event, full
diet, craft specialization, or site activity map.

## Core Rule

Lithic use-wear and residue claims must keep artefact, edge, raw material,
taphonomy, cleaning, trace formation, residue assignment, experiment, action,
task scale, and public memory separate.

## Required Controls

- Lithic-usewear gate.
- Source lock.
- Page-anchor audit.
- Refusal register.
- Cultural-memory register.
- Sibling-boundary register.
- Cross-case checklist.

## Decision

Promote directly to Core under rubric v1.43. Use-wear and residue papers expose
a general RUINS boundary: surface traces can support bounded functional
inference, but they cannot stand in for total task, diet, hunting, craft, or
behavioral histories.
