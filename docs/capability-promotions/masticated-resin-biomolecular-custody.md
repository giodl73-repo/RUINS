# Masticated Resin Biomolecular Custody

**Capability id:** `ruins-capability:masticated-resin-biomolecular-custody`
**Promotion:** Candidate -> Active -> Core
**Trigger paper:** `research/publications/panel-birch-pitch-masticated-resin-custody/`
**Rubric:** v1.39

## Problem

Chewed birch pitch can preserve ancient human DNA, oral microbiome signal, food
DNA, tooth-use traces, and technological context. Its vivid public image as
"ancient chewing gum" makes it easy to collapse a small masticated object into a
complete person, exact last meal, disease diagnosis, medical treatment, or whole
Stone Age lifestyle.

## Core Rule

Masticated resin claims must keep object, use, chewing event, preservation,
sampling, authentication, molecular assignment, inference scale, tool or medical
function, and public memory separate.

## Required Controls

- Masticated-resin gate.
- Source lock.
- Page-anchor audit.
- Refusal register.
- Cultural-memory register.
- Sibling-boundary register.
- Cross-case checklist.

## Decision

Promote directly to Core under rubric v1.39. The Syltholm birch-pitch case
creates a reusable control for masticated resin, tar, pitch, gum, and comparable
chewed biomolecular objects.
