# Microbotanical Starch-Phytolith Custody

**Capability id:** `ruins-capability:microbotanical-starch-phytolith-custody`
**Promotion:** Candidate -> Active -> Core
**Trigger paper:** `research/publications/panel-microbotanical-starch-phytolith-custody/`
**Rubric:** v1.56

## Problem

Plant microremains can preserve evidence that macrobotanical remains miss, but
they are vulnerable to overpromotion. Starch grains, phytolith morphotypes,
pollen counts, dental-calculus microremains, tool residues, ceramic crusts,
sediment microfossils, reference-collection matches, processing damage,
abundance counts, and public ancient-meal stories can be collapsed into exact
species, complete diet, direct meals, domestication proof, crop-adoption dates,
tool function, cooking recipes, medicine, craft processes, storage systems,
seasonal calendars, or complete plant-use histories.

## Core Rule

Microbotanical claims must keep sample, microremain type, preservation,
contamination, extraction, reference collection, taxonomic assignment,
processing damage, context, residue formation, quantification, activity
inference, diet/domestication synthesis, and public plant-use memory separate.

## Required Controls

- Microbotanical-residue gate.
- Source lock.
- Page-anchor audit.
- Refusal register.
- Cultural-memory register.
- Sibling-boundary register.
- Cross-case checklist.

## Decision

Promote directly to Core under rubric v1.56. Microbotanical starch-phytolith
custody creates a reusable control for plant microremain, contamination,
taxonomic assignment, processing, diet, domestication, and public meal claims.
