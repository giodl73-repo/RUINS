# OSL Sediment-Age Custody

**Capability id:** `ruins-capability:osl-sediment-age-custody`
**Promotion:** Candidate -> Active -> Core
**Trigger paper:** `research/publications/panel-madjedbebe-osl-sediment-age-custody/`
**Rubric:** v1.51

## Problem

Optically stimulated luminescence can date sediment burial around archaeology,
but a sediment age is not an exact artefact age or migration event. Single-grain
ages, dose distributions, bleaching assumptions, dose-rate models, stratigraphic
bands, artefact refits, and public first-arrival stories can be collapsed into
exact arrival dates, certain in-situ association, migration routes, megafauna
extinction causes, genetic history, or complete continent-scale peopling models.

## Core Rule

OSL claims must keep sample, grain population, light exposure, burial context,
dose model, dose-rate environment, stratigraphy, artefact association, age
model, occupation inference, regional synthesis, and public first-arrival memory
separate.

## Required Controls

- OSL sediment-age gate.
- Source lock.
- Page-anchor audit.
- Refusal register.
- Cultural-memory register.
- Sibling-boundary register.
- Cross-case checklist.

## Decision

Promote directly to Core under rubric v1.51. The Madjedbebe case creates a
reusable control for luminescence-dated sediments, occupation timing, and
first-arrival public memory.
