# Parchment Biocodicology Custody

**Capability id:** `ruins-capability:parchment-biocodicology-custody`
**Promotion:** Candidate -> Active -> Core
**Trigger paper:** `research/publications/panel-parchment-biocodicology-custody/`
**Rubric:** v1.42

## Problem

Biocodicology reads manuscripts as biological artefacts, using eraser crumbs,
ZooMS, collagen peptides, DNA, mtDNA, and microbiome signals. These methods can
identify animal species and material histories, but they can also be overread as
complete provenance, scriptorium attribution, trade route, herd history, library
reconstruction, textual authority, or conservation status.

## Core Rule

Parchment biocodicology claims must keep object, folio, sampling, conservation,
material, molecular assignment, quire/document scale, production inference,
textual boundary, and public memory separate.

## Required Controls

- Parchment-biocodicology gate.
- Source lock.
- Page-anchor audit.
- Refusal register.
- Cultural-memory register.
- Sibling-boundary register.
- Cross-case checklist.

## Decision

Promote directly to Core under rubric v1.42. The eraser-crumb and ZooMS
parchment cases create reusable controls for manuscript biomolecular evidence
without confusing material substrate with textual or institutional authority.
