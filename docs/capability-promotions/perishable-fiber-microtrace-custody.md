# Perishable Fiber Microtrace Custody

**Capability id:** `ruins-capability:perishable-fiber-microtrace-custody`
**Promotion:** Candidate -> Active -> Core
**Trigger paper:** `research/publications/panel-dzudzuana-flax-fiber-custody/`
**Rubric:** v1.44

## Problem

Perishable fiber technologies rarely survive as complete objects. Microscopic
fibers, twist, cut ends, knots, color, dye signals, hair, palynology, and
sediment context can preserve traces of cordage, basketry, sewing, hafting, or
cloth-related practices, but these traces are easily overread as complete
textiles, garments, woven cloth, craft specialization, trade networks, or
oldest-clothing stories.

## Core Rule

Perishable fiber claims must keep sample, layer, fiber identity, preservation,
contamination, modification, color, technique, object inference, activity scale,
and public memory separate.

## Required Controls

- Perishable-fiber gate.
- Source lock.
- Page-anchor audit.
- Refusal register.
- Cultural-memory register.
- Sibling-boundary register.
- Cross-case checklist.

## Decision

Promote directly to Core under rubric v1.44. The Dzudzuana flax-fiber case
creates a reusable control for microscopic textile and cordage traces without
turning partial fibers into complete perishable objects or social histories.
