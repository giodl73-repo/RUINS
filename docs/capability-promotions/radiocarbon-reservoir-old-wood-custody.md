# Radiocarbon Reservoir Old-Wood Custody

**Capability id:** `ruins-capability:radiocarbon-reservoir-old-wood-custody`
**Promotion:** Candidate -> Active -> Core
**Trigger paper:** `research/publications/panel-radiocarbon-reservoir-old-wood-custody/`
**Rubric:** v1.55

## Problem

Radiocarbon dating is foundational, but its outputs can be overpromoted.
Radiocarbon ages, calibrated ranges, reservoir corrections, Delta-R values,
freshwater hardwater offsets, marine diet corrections, old-wood charcoal dates,
Bayesian boundaries, outlier rejections, summed probability distributions,
wiggle matches, food-crust dates, and public calendar-date stories can be
collapsed into exact event dates, direct occupation dates, burial dates,
transition boundaries, population curves, migration proof, collapse timing,
diet proof, pottery-use dates, or solved chronologies.

## Core Rule

Radiocarbon chronology claims must keep sample material, organism life, carbon
source, context, pretreatment, measurement, calibration curve, reservoir model,
old-wood risk, dietary offset, Bayesian prior, model output, event inference,
and public date memory separate.

## Required Controls

- Radiocarbon-reservoir gate.
- Source lock.
- Page-anchor audit.
- Refusal register.
- Cultural-memory register.
- Sibling-boundary register.
- Cross-case checklist.

## Decision

Promote directly to Core under rubric v1.55. Radiocarbon reservoir old-wood
custody creates a reusable control for calibrated dates, reservoir corrections,
Bayesian models, SPDs, wiggle matches, and public exact-date claims.
