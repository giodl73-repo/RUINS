## Overclaim Pressure

Publication-ready status can be misread as permission to make every adjacent
exact claim: primary-source quotation, edition-page reading, translation, route,
specification, reconstruction, or destruction ledger. RUINS needs a faster
promotion path where a paper can publish a bounded method argument while exact
claim surfaces remain explicitly refused.

## Scope Decision

Current paper permission:

- Allowed: source-scoped method claims, custody-ladder claims, institutional
  memory boundaries, and named evidence-layer distinctions.
- Refused: exact quotation, exact reading, exact translation, exact source-page,
  exact reconstruction, whole-corpus, and literal catastrophe-ledger claims
  without dedicated custody.
- Future P3: direct primary/edition/page custody, exact textual work,
  translation audit, route/name/specification/reconstruction audit when needed.
- Sibling handoff: FONTES / MUNDUS for source editions and quotations; CANON
  for identity surfaces; MAXIM / LUCIA for broader history.
