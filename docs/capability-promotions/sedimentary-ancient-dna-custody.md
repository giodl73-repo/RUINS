# Sedimentary Ancient DNA Custody

**Capability id:** `ruins-capability:sedimentary-ancient-dna-custody`
**Promotion:** Candidate -> Active -> Core
**Trigger paper:** `research/publications/panel-denisova-sedimentary-dna-custody/`
**Rubric:** v1.45

## Problem

Sedimentary ancient DNA can reveal hominin and faunal presence where fossils are
scarce, but cave sediment is not a direct census. Grid samples, layers,
microstratigraphic hotspots, mtDNA, nuclear DNA, faunal DNA, bone/fecal
particles, and artefact contexts can be collapsed into exact occupants,
continuous occupation, direct species encounters, route histories, artefact-maker
identity, or absence proof.

## Core Rule

Sedimentary DNA claims must keep sample, stratigraphy, depositional process,
spatial grid, DNA source, authentication, taxon/lineage assignment, association
with artefacts/remains, temporal inference, absence claim, and public memory
separate.

## Required Controls

- Sedimentary-DNA gate.
- Source lock.
- Page-anchor audit.
- Refusal register.
- Cultural-memory register.
- Sibling-boundary register.
- Cross-case checklist.

## Decision

Promote directly to Core under rubric v1.45. The Denisova Cave sedimentary-DNA
case creates a reusable control for cave sediment DNA, microstratigraphic
hotspots, and no-bones-needed occupation-history claims.
