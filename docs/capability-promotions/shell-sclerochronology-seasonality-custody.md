# Shell Sclerochronology Seasonality Custody

**Capability id:** `ruins-capability:shell-sclerochronology-seasonality-custody`
**Promotion:** Candidate -> Active -> Core
**Trigger paper:** `research/publications/panel-shell-midden-sclerochronology-seasonality-custody/`
**Rubric:** v1.52

## Problem

Shell sclerochronology can reconstruct harvest season and nearshore conditions,
but a shell is not a complete occupation calendar. Growth increments, oxygen
isotope values, shell-edge samples, modern baselines, clumped-isotope
temperatures, species ecology, and midden counts can be collapsed into exact
harvest days, permanent settlement, complete diet, mobility routes, climate
records, resource depletion, or whole coastal lifeways.

## Core Rule

Shell seasonality claims must keep specimen, growth sequence, edge preservation,
isotope system, habitat, baseline, diagenesis, processing, death-season model,
assemblage sampling, occupation inference, and public seasonal-lifeway memory
separate.

## Required Controls

## Decision

Promote directly to Core under rubric v1.52. Shell midden sclerochronology
creates a reusable control for seasonal harvesting, coastal occupation, and
nearshore-climate proxy claims.
