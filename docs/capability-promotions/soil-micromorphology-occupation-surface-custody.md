# Soil Micromorphology Occupation-Surface Custody

**Capability id:** `ruins-capability:soil-micromorphology-occupation-surface-custody`
**Promotion:** Candidate -> Active -> Core
**Trigger paper:** `research/publications/panel-soil-micromorphology-occupation-surface-custody/`
**Rubric:** v1.58

## Problem

Soil micromorphology can expose floors, hearths, dung, plaster, trampling,
microstratigraphy, and activity residues that are invisible at excavation scale.
Those traces can be overpromoted. Thin sections, microstratigraphic laminae,
floor sediments, ash lenses, dung spherulites, phytolith concentrations, plaster
layers, trampled deposits, soil chemistry spots, FTIR signals, image-analysis
clusters, and public household scenes can be collapsed into exact activity
areas, complete room functions, daily-life reconstructions, household
biographies, occupation intensity, abandonment episodes, cleaning routines,
craft workshops, storage areas, penning systems, ritual deposits, or complete
site-use maps.

## Core Rule

Micromorphology claims must keep block sample, context, microfacies, sediment
source, depositional pathway, post-depositional alteration, activity residue,
floor renewal, cleaning/removal, feature boundary, spatial sampling, analogue,
behavior inference, and public domestic memory separate.

## Decision

Promote directly to Core under rubric v1.58. Soil micromorphology
occupation-surface custody creates a reusable control for floor, hearth, dung,
activity-area, room-function, and public household claims.
