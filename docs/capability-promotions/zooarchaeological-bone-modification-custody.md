# Zooarchaeological Bone-Modification Custody

**Capability id:** `ruins-capability:zooarchaeological-bone-modification-custody`
**Promotion:** Candidate -> Active -> Core
**Trigger paper:** `research/publications/panel-zooarchaeological-bone-modification-custody/`
**Rubric:** v1.57

## Problem

Bone surface modification and mortality profiles can identify important
taphonomic and behavioral signals, but they are easy to overpromote. Cut marks,
tooth marks, trampling striations, percussion scars, burned bone, gnawing marks,
mortality profiles, age-at-death classes, skeletal part frequencies, NISP/MNI
counts, 3D mark shapes, and public meat-eating stories can be collapsed into
exact butchery events, certain hunting or scavenging behavior, complete diet,
ritual feasts, species-specific predators, tool types, social organization,
kill-off strategies, seasonality, site function, human-evolution proof, or
complete subsistence systems.

## Core Rule

Zooarchaeological bone-modification claims must keep specimen, taxon, skeletal
element, surface preservation, mark type, actor, formation process, timing,
assemblage sampling, age profile, quantification, experimental comparison,
behavior inference, and public subsistence memory separate.

## Required Controls

- Bone-modification gate.
- Source lock.
- Page-anchor audit.
- Refusal register.
- Cultural-memory register.
- Sibling-boundary register.
- Cross-case checklist.

## Decision

Promote directly to Core under rubric v1.57. Zooarchaeological
bone-modification custody creates a reusable control for cut marks, tooth marks,
trampling, percussion, mortality profiles, and public subsistence claims.
