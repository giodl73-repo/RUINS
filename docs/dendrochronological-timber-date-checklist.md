# Dendrochronological Timber-Date Checklist

Use this checklist when construction date, remodeling date, occupation span,
abandonment, drought stress, timber trade, building sequence, community size,
forest management, climate reconstruction, or public exact-year memory is
reconstructed from tree rings, beams, posts, charcoal, sapwood, waney edge, bark
edge, crossdating, reference chronologies, provenance signatures, reused
timbers, old wood, repair episodes, or cutting clusters.

## Required Separations

| Surface | Required custody question |
|---|---|
| Sample | Which beam, post, charcoal fragment, room, layer, collection, and lab record is measured? |
| Tree | What species, growth habitat, woodland source, and biological variability are locked? |
| Ring sequence | Is the sequence complete, crossdated, complacent, missing, distorted, or short? |
| Terminal ring | Is bark edge, waney edge, sapwood, heartwood, or nonterminal outer ring preserved? |
| Felling | Is the claim tree growth year, felling year, transport episode, stockpiling, or use date? |
| Reuse | Are old wood, curated beams, repair timber, scavenging, and secondary placement separated? |
| Construction | Is the claim construction start, completion, remodeling, roof repair, or room-use phase? |
| Chronology | Which reference chronology, match quality, and uncertainty support the date? |
| Provenance | Is timber origin local, nonlocal, traded, floated, curated, or chemically inferred? |
| Synthesis | Are occupation, abandonment, drought, community size, and migration handed to siblings? |
| Memory | Are exact-year, cliff-dwelling, drought-collapse, and vanished-community stories contained? |

## Refuse Unless Separately Locked

- Terminal ring is exact building completion.
- Felling cluster is a complete construction sequence.
- A timber date is a full occupation span.
- Last dated beam is abandonment.
- Drought rings prove migration or collapse.
- Beam counts are community size.
- Timber provenance is a complete trade map.
- Reused wood is direct room-use chronology.
