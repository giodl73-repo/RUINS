# Dispersed Object Provenance Checklist

Status: core RUINS method
Capability: `ruins-capability:dispersed-object-provenance-custody`
Owner: ASHLAR / PALIMPSEST / ERRATA

Use this checklist when objects materially survive but their ruin condition is
broken custody: removal, dispersal, museum holding, contested title, legal
transfer, loan, return, digital record, replica, display, or living relation.

## Required Separation

| Layer | Required question | RUINS treatment |
|---|---|---|
| Object identity | Which object, fragment, group, material, and source label is being discussed? | Object-specific; group labels do not settle every object. |
| Original context | What monument, palace, shrine, site, ritual, or social setting is source-locked? | Context claim only at the audited scope. |
| Removal / dispersal | What event, purchase, excavation, expedition, sale, transfer, or market path moved the object? | Provenance layer, not legal closure. |
| Current holding | Which institution or collection currently holds, displays, stores, or documents it? | Current custody layer only. |
| Legal / governance action | Is there deaccession, title transfer, loan, act of parliament, request, mediation, or return? | Source-specific legal/governance layer. |
| Physical movement | Has the object actually moved, or only title/loan/request/status changed? | Separate physical custody from legal status. |
| Digital / replica display | Is the record, image, cast, plaster copy, or digital catalogue standing in for the object? | Mediated access layer, not object equivalent. |
| Living / public memory | What community, national, ritual, heritage, or restitution meanings attach to the object? | Afterlife/custody layer, not proof of every object detail. |

## Mandatory Refusals

A dispersed-object paper must refuse:

1. one object's provenance as the whole corpus provenance;
2. current possession as original context;
3. legal title as ethical closure;
4. return, loan, transfer, display, and digital record as interchangeable;
5. replica, cast, plaster copy, or digital image as object-equivalent;
6. public memory as object-level provenance;
7. complete restitution or complete reunification without object-level custody.

## Cross-Case Test

| Test | Benin Bronzes | Parthenon Sculptures | Core rule |
|---|---|---|---|
| Custody break | 1897 British expedition, palace/shrine looting, global dispersal, partial returns. | Elgin removal from Parthenon/Acropolis, British Museum acquisition, Greek return request, split display. | Removal and current holding are separate layers. |
| Object group pressure | Broad "Bronzes" label spans materials, object types, provenance paths. | "Parthenon Sculptures" spans frieze, metopes, pediments, fragments, plaster copies, and museum groupings. | Group labels require object-level caution. |
| Reconnection layer | Digital Benin links object data, archives, oral history, provenance, maps. | Acropolis Museum aligns gallery to the monument and uses plaster copies for absent fragments. | Digital/replica/recontextual display is access, not object equivalence. |
| Overclaim pressure | Return of some objects repairs whole corpus. | One museum's title/display argument settles reunification. | Refuse total repair, total title closure, and total context restoration. |

## Automatic Cap

A paper cannot score above `34` if it treats current possession, legal title,
loan, return, replica, digital record, or display as complete provenance,
original context, restitution, or reunification without this checklist or an
equivalent dispersed-object custody gate.
