# Geophysical Subsurface Survey Checklist

Use this checklist when buried buildings, street grids, walls, ditches, pits,
tombs, postholes, hearths, kilns, pipes, water systems, monuments, urban plans,
sacred landscapes, excavation targets, preservation status, or public
city-without-digging memory are reconstructed from GPR, magnetometry, magnetic
susceptibility, resistivity, electromagnetic induction, conductivity,
gradiometry, radar time slices, amplitude maps, depth slices, anomaly clusters,
interpolation, survey transects, motorized arrays, or integrated geophysical
datasets.

## Required Separations

| Surface | Required custody question |
|---|---|
| Instrument | Which sensor, array, frequency, spacing, and survey mode produced the record? |
| Grid | What survey grid, transect spacing, navigation, and georeferencing control the map? |
| Processing | Which filtering, interpolation, slicing, correction, and visualization choices were applied? |
| Depth model | Does the claim depend on assumed velocity, depth conversion, or slice thickness? |
| Soil contrast | Are geology, moisture, fill, burning, stone, metal, and modern disturbance separated? |
| Anomaly class | Is the signal wall, ditch, pit, pipe, tomb, floor, natural feature, or unknown anomaly? |
| Ground truth | Was the anomaly checked by excavation, coring, legacy plan, surface evidence, or repeat method? |
| Chronology | Is the feature dated independently, or only mapped by subsurface contrast? |
| Function | Are building, street, ritual, hydraulic, domestic, industrial, and natural explanations separated? |
| Plan synthesis | Are confirmed features, candidate alignments, interpolation, and complete-plan claims separated? |
| Memory | Are whole-city, hidden-monument, nondestructive certainty, and sacred-landscape stories contained? |

## Refuse Unless Separately Locked

- GPR time slice is confirmed building.
- Magnetometry plot is complete street grid.
- Anomaly cluster is certain tomb or ritual feature.
- Depth slice is final chronology.
- Interpolated grid is exact wall line.
- Nondestructive survey is excavation-equivalent proof.
- Integrated map is whole city.
- Hidden-monument image is solved sacred landscape.
