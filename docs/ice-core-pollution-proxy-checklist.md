# Ice-Core Pollution Proxy Checklist

Use this checklist when mining, smelting, economy, plague, war, imperial
expansion, industrial output, public health, or civilizational change is
reconstructed from ice cores, lead concentrations, lead isotopes, aerosols, dust,
annual layers, deposition flux, atmospheric transport models, source
apportionment, or historical event alignment.

## Required Separations

| Surface | Required custody question |
|---|---|
| Core | Which core, depth, site, archive, sample resolution, and handling history produced the signal? |
| Chronology | Which annual layer, age model, uncertainty, tie point, and interpolation supports the date? |
| Contaminant | Is the signal lead, isotope ratio, dust, sea salt, volcanic input, wildfire aerosol, or mixed? |
| Deposition | What snow accumulation, post-depositional movement, seasonality, and flux controls are locked? |
| Source | Which source region, ore field, smelter class, natural background, and isotope match is claimed? |
| Transport | Which atmospheric model, wind field, receptor assumption, and uncertainty supports source travel? |
| Conversion | Is the inference concentration, deposition flux, emissions estimate, industrial output, or economy? |
| Event | Is the alignment war, plague, policy, expansion, recession, or only chronological proximity? |
| Exposure | Are atmospheric concentration, inhalation, blood lead, health, and population models separate? |
| Memory | Are Rome-pollution, IQ-loss, fall-of-empire, and economic-dashboard stories contained? |

## Refuse Unless Separately Locked

- Ice-core lead is exact mining output.
- Deposition flux is a precise economic index.
- A pollution drop proves one war, plague, or policy event.
- Transport modelling is complete source certainty.
- Lead pollution is population exposure or IQ loss without exposure modelling.
- Roman pollution is civilizational decline.
- A proxy curve is complete industrial or economic history.
