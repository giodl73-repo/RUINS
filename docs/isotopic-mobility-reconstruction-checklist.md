# Isotopic Mobility Reconstruction Checklist

Use this checklist when origin, movement, diet, or object provenance is
reconstructed from strontium, oxygen, carbon, nitrogen, sulfur, lead, or related
isotope evidence.

## Required Separations

| Surface | Required custody question |
|---|---|
| Sample | Which tooth, bone, dentine, enamel, hair, nail, textile, animal, plant, soil, or object was sampled? |
| Window | What life stage, season, production interval, or material source does it record? |
| Preservation | Could diagenesis, contamination, conservation, liming, or modern treatment affect the value? |
| Baseline | Which local baseline, isoscape, map, or comparison range defines local/non-local? |
| Comparison | Are human, animal, textile, object, soil, and regional values compared separately? |
| Mobility | Is the claim local/non-local, possible region, repeated movement, route, or residence? |
| Identity | Are ethnicity, lineage, status, named origin, and biography separately sourced? |
| Memory | Is public origin language handled as reception, not final archaeology? |

## Refuse Unless Separately Locked

- Isotope values prove exact birthplace.
- Baselines are route maps.
- Non-local means ethnicity or national identity.
- Textile or object provenance proves personal travel.
- A public origin story is a final archaeological conclusion.
