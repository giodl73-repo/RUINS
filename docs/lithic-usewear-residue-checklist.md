# Lithic Use-Wear Residue Checklist

Use this checklist when tool function, worked material, hafting, task, skill,
site activity, diet, hunting, craft, or behavior is reconstructed from edge
damage, polish, striations, rounding, residue morphology, starch, phytolith,
blood, collagen, FTIR, microscopy, surface texture, or experimental replicas.

## Required Separations

| Surface | Required custody question |
|---|---|
| Artefact | Which tool, assemblage, raw material, context, date, and edge are under claim? |
| Edge | Is the trace edge damage, polish, rounding, striation, micro-scarring, residue, or mixed evidence? |
| Raw material | How does stone texture, fracture, grain, weathering, and heterogeneity affect trace formation? |
| Taphonomy | What post-depositional, handling, curation, or conservation processes altered the surface? |
| Cleaning | Were washing, brushing, solvents, storage, and contamination risks documented? |
| Trace formation | Is the trace use, hafting, prehension, manufacture, retouch, trampling, or residue transfer? |
| Residue assignment | Are starch, phytolith, blood, collagen, plant, animal, and mineral calls separately graded? |
| Experiment | Which replica, worked material, motion, duration, force, and reference collection supports analogy? |
| Action | Is the inference cutting, scraping, drilling, hide work, plant work, bone work, hafting, or mixed use? |
| Scale | Is the claim edge-level, tool-level, assemblage-level, site activity, or social behavior? |

## Refuse Unless Separately Locked

- Edge polish is exact tool function.
- A residue morphology is a single worked species.
- An experimental analogue is a complete task sequence.
- A hafting trace is a full hafting system.
- Blood or collagen residue is a hunting event.
- Plant residue is a complete diet.
- Use-wear is craft specialization or skill biography.
