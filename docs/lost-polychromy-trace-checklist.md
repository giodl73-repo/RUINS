# Lost Polychromy Trace Checklist

Use this checklist when lost color is known through visible pigment, imaging,
sampling, conservation cleaning, pigment identification, reconstruction, digital
display, or public color memory.

## Required Separations

| Surface | Required custody question |
|---|---|
| Visible pigment | Is color directly visible, condition-dependent, or only visible under certain lighting? |
| Imaging output | Which method produced the signal: VIL, XRF, microscopy, multiband imaging, or other analysis? |
| Sample or chemical result | Does the result identify pigment type, binding context, distribution, or only local trace? |
| Conservation cleaning | Did cleaning reveal color, alter visibility, or only remove dirt from a specific surface? |
| Color reconstruction | Which parts are directly supported, inferred, comparative, or speculative? |
| Exhibition or digital access | Is the public seeing an original, reconstruction, projection, AR layer, or research visualization? |
| Meaning and function | Is color being used to infer divine, funerary, aesthetic, or social meaning beyond the trace? |
| White-marble memory | Is later reception handled as afterlife rather than evidence of original appearance? |

## Refuse Unless Separately Locked

- A pigment trace proves the complete original color scheme.
- Imaging output is direct access to final appearance.
- A pigment name supplies complete painterly method.
- Cleaning or reconstruction restores the original object.
- AR, projection, or exhibition images are object-equivalent.
- Color evidence settles final ritual, divine, or aesthetic meaning.
- One object's color evidence defines a whole culture's aesthetic.
