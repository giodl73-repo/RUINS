# Masticated Resin Biomolecular Checklist

Use this checklist when genome, oral microbiome, diet, disease, technology,
medicine, or identity is reconstructed from chewed pitch, resin, tar, gum, tooth
marks, saliva, ancient DNA, microbial DNA, or food DNA.

## Required Separations

| Surface | Required custody question |
|---|---|
| Object | Which piece of pitch, resin, tar, or gum was sampled, and what is its context? |
| Use | Is the object adhesive, masticated material, repair medium, medicine, fuel byproduct, or unknown? |
| Chewing event | Are tooth marks, saliva, handling, or repeated chewing separately evidenced? |
| Preservation | How did resin chemistry, burial, handling, and conservation affect biomolecule survival? |
| Sampling | Which extraction, contamination, and destructive-sampling controls are locked? |
| Authentication | Are ancient DNA damage, coverage, microbial profile, and contamination separated? |
| Assignment | Are human, microbial, viral, plant, animal, and environmental DNA assignments graded separately? |
| Inference scale | Is the claim individual, momentary exposure, recent meal, population pattern, or technology history? |
| Function | Are adhesive, antiseptic, medicinal, and toolmaking claims separately sourced? |
| Memory | Are public chewing-gum, named-person, last-meal, and lifestyle stories kept separate? |

## Refuse Unless Separately Locked

- A chewed resin object is a complete person.
- Food DNA is an exact last meal.
- Microbial or viral DNA is a disease diagnosis.
- Pitch chewing is direct medical treatment.
- Birch pitch use is a complete toolmaking system.
- One object's DNA is population history.
- Ancient chewing-gum memory is a full Stone Age lifestyle.
