# Microbotanical Starch-Phytolith Checklist

Use this checklist when plant use, diet, domestication, crop adoption, tool
function, food processing, cooking, storage, medicine, craft, seasonality,
residue origin, or public ancient-meal memory is reconstructed from starch
grains, phytoliths, pollen, microbotanical residues, dental calculus, stone
tools, grinding stones, ceramics, sediments, soil samples, carbonized residues,
reference collections, microscopy, morphotypes, damage patterns, or abundance
counts.

## Required Separations

| Surface | Required custody question |
|---|---|
| Sample | Which artefact, calculus, sediment, crust, tool, layer, collection, and lab record is sampled? |
| Microremain | Is the signal starch, phytolith, pollen, fiber, mixed residue, or modern contaminant? |
| Preservation | What survival, degradation, burning, cooking, burial, and soil chemistry affect the signal? |
| Contamination | What modern plant, lab, handling, sediment, survey, and control samples bound contamination? |
| Extraction | Which protocol, microscope, slide, count, image, and observer procedure is locked? |
| Reference | Which reference collection, taxonomic criteria, and morphotype uncertainty support identification? |
| Assignment | Is the claim family, genus, species, crop/wild status, plant part, or unresolved type? |
| Processing | Are grinding, boiling, roasting, charring, chewing, storage, and use-wear separated? |
| Quantification | Are counts presence/absence, abundance, percentage, density, or biased recovery? |
| Synthesis | Are diet, domestication, tool function, cooking, medicine, and craft handed to siblings? |
| Memory | Are direct meal, first crop, ancient recipe, and complete plant-use stories contained? |

## Refuse Unless Separately Locked

- Starch grain is exact species.
- Phytolith morphotype is domestication proof.
- Tool residue is complete tool function.
- Dental calculus microremain is direct meal.
- Sediment microfossil is crop-adoption date.
- Abundance count is complete diet.
- Processing damage is cooking recipe.
- Plant residue is medicine or craft process.
