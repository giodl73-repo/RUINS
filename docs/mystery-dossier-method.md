# Mystery Dossier Method

RUINS can investigate famous mysteries when the question is framed as damaged
knowledge reconstruction, not hidden certainty. A mystery dossier links clues
across physical traces, texts, copies, catalogues, maps, measurements,
reconstructions, and public memory while preserving the custody status of each
link.

## Purpose

The dossier answers:

1. What is the mystery claim?
2. What evidence actually survives?
3. Which clues are direct, copied, translated, reconstructed, reported, or
   legendary?
4. Which links strengthen the reconstruction?
5. Which links only strengthen the story?
6. What remains unknown after responsible reconstruction?

## Required Sections

| Section | Required output |
|---|---|
| Mystery claim | One bounded question, not a broad lore topic. |
| Evidence inventory | Surviving objects, contexts, inscriptions, texts, datasets, copies, catalogues, maps, images, and later reports. |
| Clue graph | A link table from source to claim, with custody status for each link. |
| Link strength | Direct, corroborated, plausible, weak, legendary, or modern-projection. |
| Failure points | Where the chain depends on lost originals, copies, translations, later summaries, restoration, sampling, or public memory. |
| Alternative explanations | Competing readings that fit the same surviving evidence. |
| Reconstruction boundary | What can be responsibly pieced together and what must remain unresolved. |
| Mystery pressure | Why the case attracts overclaim, certainty, tourism, nationalism, treasure language, or media compression. |
| Reusable capability | The gate, register, checklist, or cap the dossier should test or create. |

## Clue Graph Table

| Link | Source | Custody status | Claim supported | Strength | Failure mode |
|---|---|---|---|---|---|
| L1 | Physical trace, text, copy, catalogue, image, or report | Original, fragment, copy, translation, reconstruction, later witness, or public memory | Exact claim the link can support | Direct, corroborated, plausible, weak, legendary, or projection | What would break if this link is overpromoted |

## Candidate Dossier Queue

| Dossier | Core mystery | Primary custody pressure |
|---|---|---|
| Herculaneum unread texts | What can virtual unwrapping recover from carbonized scrolls? | Mediated recovery, OCR/HTR, edition drift, public recovered-text memory. |
| Linear A and undecipherment | What can be known without a decipherment? | Non-decipherment discipline, sign corpus custody, false translation control. |
| Voynich manuscript claims | What do material, codicological, linguistic, and provenance clues actually support? | Citation-chain fragility, undeciphered text pressure, modern hoax/solution claims. |
| Atlantis as reception problem | Which ancient text claims exist, and what later geography/lore adds? | Source chain custody, myth-to-site overclaim, public memory containment. |
| Greek fire transmission | What recipe, tactical, and technology claims survive? | Lost technique transmission, later summary dependence, experimental reconstruction boundary. |
| Roman concrete recovery | Which ancient technique claims are genuinely lost, continuous, or experimentally reconstructed? | Partial technical transmission, material trace, recipe overclaim. |
| Lost Library of Alexandria | What was lost, when, and how did later singular-destruction memory form? | Famous loss compression, institutional memory boundary, destroyed corpus custody. |
| Tomb or relic identity claims | When can remains, inscriptions, context, and later tradition identify a person? | Identity claim caps, biometric/genomic custody, public memory pressure. |

## Mandatory Refusals

A RUINS mystery dossier must refuse:

1. a mystery label as evidence of suppression;
2. absence of evidence as proof of hidden survival;
3. resemblance as direct identification;
4. later citation chains as original testimony;
5. reconstructed text, recipe, site, body, or object as the original;
6. media fame as research significance;
7. a single elegant solution when multiple weak links remain;
8. treasure-hunting, site-location, or recovery instructions.

## Success Test

A good dossier makes the mystery sharper after removing weak certainty. It should
leave a reader with a clearer map of clues, stronger and weaker links, live
alternatives, and a responsible statement of what can be pieced together.
