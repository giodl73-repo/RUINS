# Obsidian Provenance-Hydration Checklist

Use this checklist when source, exchange, mobility, chronology, trade route,
tool production, procurement, interaction sphere, ethnicity, territoriality,
site connection, or public network memory is reconstructed from obsidian
artefacts, flakes, blades, geochemical fingerprints, XRF, pXRF, INAA,
LA-ICP-MS, source libraries, sub-sources, hydration rims, diffusion rates,
temperature history, intrinsic water, cortex, reduction stage, or source-distance
models.

## Required Separations

| Surface | Required custody question |
|---|---|
| Artefact | Which artefact, flake, blade, core, layer, collection, and context is measured? |
| Glass source | Which geological source, sub-source, quarry, flow, or source library is being matched? |
| Instrument | Is the method XRF, pXRF, INAA, LA-ICP-MS, visual sourcing, or mixed evidence? |
| Calibration | What standards, cross-calibration, elements, thresholds, and uncertainty support assignment? |
| Assignment | Is the claim source, likely source, nonmatch, sub-source, or unresolved cluster? |
| Hydration | Which surface, rim measurement, intrinsic water, diffusion model, and thermal history are locked? |
| Context | Is the obsidian in primary use, discard, curated, residual, traded, collected, or redeposited context? |
| Reduction | Are cortex, flake size, blade/core stage, use-wear, and workshop inference separated? |
| Exchange | Is the claim procurement, down-the-line exchange, direct trade, mobility, or political network? |
| Synthesis | Are route, market, ethnicity, territoriality, alliance, and interaction sphere handed to siblings? |
| Memory | Are ancient-trade, migration, origin, and network-map stories contained? |

## Refuse Unless Separately Locked

- Source assignment is a direct trade route.
- Hydration rim is exact artefact age.
- Source distance is migration path.
- Obsidian cluster is ethnic identity.
- Flake counts are market scale.
- Cortex pattern is complete procurement system.
- Sub-source match is territorial boundary.
- Exchange map is a political alliance.
