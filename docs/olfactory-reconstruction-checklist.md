# Olfactory Reconstruction Checklist

Use this checklist when a lost smell is reconstructed from residues, containers,
workshops, recipes, ingredient identifications, paleobotany, experimental
batches, conservation condition, synthetic smellscapes, or exhibition scents.

## Required Separations

| Surface | Required custody question |
|---|---|
| Trace | Is the claim based on residue, vessel odor, workshop context, text, or later conservation smell? |
| Container | Which jar, bottle, vessel, body, room, or object mediated the odor evidence? |
| Ingredient | Are compounds, botanical matches, resins, oils, fats, bitumen, or substitutes separately identified? |
| Recipe | Is the recipe textual, inferred from residue, experimental, comparative, or public-facing? |
| Aging | Have evaporation, oxidation, storage, microbial activity, and conservation changed the smell? |
| Experiment | Which modern choices shaped the batch, concentration, process, and carrier? |
| Display | Is the public smelling a reconstruction, synthetic smellscape, scented card, diffuser, or interpretation? |
| Memory | Is a named person, celebrity frame, afterlife claim, or "smell the past" hook handled as reception? |

## Refuse Unless Separately Locked

- A reconstruction is the exact ancient smell.
- Residue analysis is a complete recipe.
- A modern batch is a named person's personal scent.
- A museum smell is the original atmosphere.
- Current object odor is unchanged ancient odor.
- Ingredient identification supplies exact quantities or process.
- Public "smell the past" language proves direct sensory access.
