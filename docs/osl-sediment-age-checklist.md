# OSL Sediment-Age Checklist

Use this checklist when occupation date, artefact age, first arrival, migration
route, extinction overlap, site integrity, sedimentation history, or continental
peopling is reconstructed from optically stimulated luminescence, single-grain
OSL, quartz or feldspar grains, burial dose, equivalent dose, dose rate,
bleaching, overdispersion, stratigraphic integrity, artefact refits, or sediment
samples.

## Required Separations

| Surface | Required custody question |
|---|---|
| Sample | Which sediment sample, depth, square, layer, grain size, and archive context was dated? |
| Grain population | Is the age from single grain, aliquot, quartz, feldspar, mixed grains, or rejected populations? |
| Light exposure | Was the signal fully bleached before burial, partially bleached, mixed, or reset after disturbance? |
| Burial context | Is the dated sediment primary deposition, reworked fill, bioturbated layer, or intrusive material? |
| Dose model | Which equivalent-dose model, overdispersion, outlier rule, and uncertainty supports the age? |
| Dose rate | Which water content, radioisotope, cosmic dose, sediment chemistry, and disequilibrium controls are locked? |
| Stratigraphy | Are sediment bands, artefact refits, micromorphology, and disturbance checks separately documented? |
| Association | Is the claim sediment age, artefact association, occupation episode, or first-arrival inference? |
| Synthesis | Are migration, genetics, megafauna extinction, route, and continent-scale history handed to siblings? |
| Memory | Are 65k, first Australians, route, sea crossing, and extinction-overlap stories contained? |

## Refuse Unless Separately Locked

- OSL age is exact human arrival date.
- Sediment age is exact artefact age.
- Artefact proximity is certain in-situ association.
- Stratigraphic integrity is complete site integrity.
- First-arrival date is a migration route.
- Occupation overlap proves megafauna extinction cause.
- Public first-arrival memory is complete peopling history.
