# Paleoparasitology Latrine Checklist

Use this checklist when health, diet, sanitation, animal proximity, travel,
migration, trade, or disease ecology is reconstructed from latrines, coprolites,
pelvic sediment, parasite eggs, sedimentary ancient DNA, ELISA, microscopy,
shotgun sequencing, or pathogen markers.

## Required Separations

| Surface | Required custody question |
|---|---|
| Deposit | Is the sample latrine fill, coprolite, pelvic sediment, midden, drain, chamber pot, or mixed sediment? |
| Context | Which site, date, feature, stratigraphy, reuse, and disturbance history are locked? |
| Sampling | Were concentration, extraction, subsampling, controls, and contamination tracked? |
| Parasite stage | Is the evidence egg, cyst, larva, adult remains, DNA, protein, or immunological signal? |
| Host | Is the parasite human-specific, zoonotic, animal-hosted, environmental, or ambiguous? |
| Assignment | Are microscopy, ELISA, shotgun DNA, taxon, plant, animal, and pathogen calls graded separately? |
| Infection | Is the claim infection, exposure, environmental contamination, diet passage, or animal proximity? |
| Diet | Are plant and animal DNA treated as dietary components rather than complete diet? |
| Sanitation | Are latrine use, waste management, water contamination, and public-health systems separated? |
| Memory | Are public ancient-waste, filth, plague, and migration stories kept separate? |

## Refuse Unless Separately Locked

- A parasite egg is an individual disease diagnosis.
- Latrine DNA is a whole diet.
- A host marker is a migration route.
- A zoonotic parasite is a complete animal-management system.
- A latrine sample is a sanitation system.
- A pathogen signal is public-health history.
- Ancient-waste memory is direct social history.
