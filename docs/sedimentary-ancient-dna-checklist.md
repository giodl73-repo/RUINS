# Sedimentary Ancient DNA Checklist

Use this checklist when hominin presence, faunal turnover, occupation history,
population relation, species overlap, ecology, site use, or absence is
reconstructed from cave sediment, sedimentary ancient DNA, mtDNA, nuclear DNA,
grid samples, microstratigraphy, hotspots, bone/fecal particles, bulk sediment,
or extraction controls.

## Required Separations

| Surface | Required custody question |
|---|---|
| Sample | Which sediment sample, block, grid square, chamber, layer, or microstratigraphic unit was tested? |
| Stratigraphy | What date, layer, disturbance, mixing, redeposition, and association controls are locked? |
| Deposition | Is DNA from sediment matrix, microscopic bone, feces, body fluid, hair, skin, water movement, or unknown source? |
| Spatial grid | Does the claim depend on point sample, grid pattern, layer average, chamber pattern, or site-wide map? |
| DNA source | Is the signal mtDNA, nuclear DNA, faunal DNA, microbial DNA, environmental DNA, or mixed? |
| Authentication | Are damage, fragment length, contamination, extraction blank, and replication controls separate? |
| Assignment | Is the call species, lineage, population, individual, hybrid relation, or unresolved taxon? |
| Association | Are DNA, artefacts, hominin remains, fauna, dates, and sediment processes independently linked? |
| Time | Is the claim presence, turnover, repeated presence, continuous occupation, overlap, or absence? |
| Memory | Are no-bones-needed, cave-occupant, species-meeting, and occupation-history stories separated? |

## Refuse Unless Separately Locked

- Sediment DNA is exact individual presence.
- Layer signal is continuous occupation.
- Co-occurring signals are direct species encounter.
- Hominin DNA identifies artefact makers.
- Faunal turnover is a migration route.
- Missing DNA is absence of a species.
- Public no-bones-needed memory is complete site history.
