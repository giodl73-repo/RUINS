# Zooarchaeological Bone-Modification Checklist

Use this checklist when butchery, hunting, scavenging, carnivore activity,
marrow extraction, tool use, feasting, ritual, diet, seasonality, herd
management, kill-off strategy, site function, human evolution, or public
meat-eating memory is reconstructed from animal bones, cut marks, tooth marks,
percussion marks, trampling striations, burning, gnawing, breakage, mortality
profiles, age-at-death, skeletal part representation, NISP/MNI counts,
microscopic mark shape, 3D surface scans, or experimental analogues.

## Required Separations

| Surface | Required custody question |
|---|---|
| Specimen | Which bone, element, side, portion, context, collection, and publication is recorded? |
| Taxon | What species, size class, age class, and anatomical identification are locked? |
| Surface | Is preservation good enough to distinguish cultural, carnivore, trampling, and weathering marks? |
| Mark | Is the modification cut, scrape, percussion, tooth pit, score, trampling, burning, or breakage? |
| Actor | Is the possible actor human, carnivore, sediment, fire, tool, trampling, or unknown? |
| Timing | Is the mark peri-mortem, post-depositional, excavation damage, or modern handling? |
| Assemblage | How do recovery, screen size, fragmentation, NISP/MNI, and skeletal part bias affect inference? |
| Mortality | Are age-at-death, seasonality, herd structure, and kill-off profiles separately modelled? |
| Analogue | Which experimental, ethnoarchaeological, microscopic, or 3D comparison supports the claim? |
| Synthesis | Are diet, hunting, scavenging, ritual, site function, and human evolution handed to siblings? |
| Memory | Are meat-eating, hunter-origin, feast, and kill-site stories contained? |

## Refuse Unless Separately Locked

- Cut mark is exact butchery event.
- Tooth mark identifies species-specific predator.
- Trampling striation is tool cut.
- Percussion scar is complete marrow-extraction system.
- Mortality profile is kill-off strategy.
- NISP/MNI count is complete diet.
- Burned bone is ritual feast.
- Skeletal part frequency is site function.
