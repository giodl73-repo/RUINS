# RUINS Mystery Dossiers

Mystery dossiers are custody-aware clue maps for damaged knowledge. They are not
solution posts. Each dossier links surviving traces to famous claims, marks the
strength of each clue, and separates reconstruction from public mystery.

Use [`../../docs/mystery-dossier-method.md`](../../docs/mystery-dossier-method.md)
as the required format.

## Dossier Rule

The output should make the clue chain more legible, not more sensational. If the
best finding is that a mystery cannot be solved from surviving evidence, that is
a valid RUINS result.
