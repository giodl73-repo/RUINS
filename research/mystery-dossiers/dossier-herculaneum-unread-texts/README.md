# Herculaneum Unread Texts Dossier

Status: initial clue dossier

## Mystery Claim

What can virtual unwrapping responsibly recover from unopened or partly damaged
Herculaneum papyri, and when does recovered text become a scholarly edition
rather than a machine-readable signal?

## Why This Fits RUINS

The Herculaneum papyri are a rare case where catastrophic damage preserved a
library while also making direct access destructive. The new breakthrough is not
simply "AI read the scrolls." It is a linked custody chain:

`carbonized roll -> CT volume -> sheet segmentation -> surface flattening ->
ink detection -> letter/word reading -> papyrological review -> edition claim ->
public recovered-library memory`

RUINS can make that chain legible without downgrading the breakthrough or
overpromoting it into complete recovery of the ancient library.

## Core Finding

PHerc. 1667 can now be treated as a strong proof of end-to-end virtual
unwrapping for one surviving inner core of one scroll. It cannot by itself prove
that every unopened scroll is readable, that every detected signal is a secure
letter, that machine output is already an edition, or that the Villa library has
been intellectually recovered.

## Reusable Pressure

This dossier should test three next RUINS pressures:

1. citation-chain fragility, because public summaries will cite breakthrough
   announcements faster than papyrological editions;
2. edition-version drift, because scan outputs, transcriptions, readings, and
   scholarly editions can diverge over time;
3. mediated recovery custody, because the object is accessed through scans,
   segmentation, machine learning, and expert review rather than direct viewing.

## Refusals

RUINS refuses:

- "AI read the library" as a substitute for scroll-by-scroll, layer-by-layer
  custody;
- complete-library recovery from one scroll;
- machine ink signal as automatic text;
- preliminary transcription as final edition;
- recovered author/title evidence as complete intellectual history;
- public renaissance language as evidence strength.
