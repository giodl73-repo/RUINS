# Dung Proxy Gate

| Layer | Dung-proxy custody question |
|---|---|
| Deposit | Is the sample dung accumulation, fuel ash, plaster, floor sediment, midden, flotation residue, or mixed deposit? |
| Proxy | Is the signal spherulite, lipid biomarker, dung fungus, phytolith, micromorphology, FTIR, or chemistry? |
| Preservation | Were burning, dissolution, water, trampling, alkalinity, curation, and sampling effects controlled? |
| Host/source | Is the dung human, herbivore, livestock, wild animal, mixed, or unidentified? |
| Formation | Was dung deposited by animals in place, collected by humans, or moved as material? |
| Reuse | Was dung used as fuel, plaster preparation, floor treatment, manure, or waste? |
| Method | Which counting, imaging, chemical, or multi-proxy workflow supports the claim? |
| Activity | Is the inference animal tending, penning, herding, fuel use, foddering, grazing, or land use? |
| Scale | Is the claim feature, household, site, regional, or historical-transition scale? |
| Memory | Are earliest-herding and domestication stories held below source-locked evidence? |
