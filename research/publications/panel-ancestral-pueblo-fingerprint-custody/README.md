# Ancestral Pueblo Fingerprint Custody

**Track:** Evidence and Absence
**Status:** publication-ready
**Rubric:** v1.38
**Score:** 46/50
**Capability decision:** promote `ruins-capability:biometric-trace-reconstruction-custody` to Core

This package uses fingerprint evidence on Ancestral Pueblo pottery as the
trigger case for biometric-trace custody. It keeps ridge-density measurement,
sex-category inference, ceramic shrinkage, sample selection, labor-division
claims, and public maker stories in separate custody layers.
