# Biometric Trace Gate

| Layer | Fingerprint custody question |
|---|---|
| Trace | Which print, ridge field, sherd, vessel, and image is being measured? |
| Substrate | What clay fabric, temper, surface finish, and firing history shaped the trace? |
| Formation | Was the impression produced during making, handling, use, or later disturbance? |
| Preservation | Which prints survived smoothing, breakage, erosion, conservation, and selection? |
| Distortion | How were shrinkage, pressure, moisture, angle, and scan/image processing controlled? |
| Measurement | Which MRB, MRD, ridge-density, or model workflow generated the value? |
| Reference | Which population data support the age or sex category, and how portable is it? |
| Uncertainty | Is output a category, probability, assemblage tendency, or individual assignment? |
| Behavior | Are craft role, learning, household production, labor division, and economy separated? |
| Memory | Are public maker and gender-work stories held below source-locked inference? |
