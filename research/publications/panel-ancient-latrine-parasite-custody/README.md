# Ancient Latrine Parasite Custody

**Track:** Evidence and Absence
**Status:** publication-ready
**Rubric:** v1.40
**Score:** 46/50
**Capability decision:** promote `ruins-capability:paleoparasitology-latrine-custody` to Core

This package uses ancient DNA from latrines in Northern Europe and the Middle
East as the trigger case for paleoparasitology-latrine custody. It keeps latrine
context, parasite egg concentration, shotgun sequencing, parasite DNA,
vertebrate and plant DNA, diet components, human-animal interaction,
sanitation, disease, movement, and public ancient-waste memory separate.
