# Ancient Pathogen Genome Custody

**Track:** Evidence and Absence
**Status:** publication-ready
**Rubric:** v1.63
**Score:** 46/50
**Capability decision:** promote `ruins-capability:ancient-pathogen-genome-custody` to Core

This package uses ancient pathogen genomics and Yersinia pestis studies as the
trigger case for pathogen-genome custody. It keeps samples, skeletal contexts,
screening methods, authentication, contamination, genome coverage, taxon
assignment, strain placement, virulence markers, host relation, chronology,
epidemiological scale, historical synthesis, and public disease memory separate.
