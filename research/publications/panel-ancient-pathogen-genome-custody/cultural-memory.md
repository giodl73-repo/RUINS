# Cultural Memory

Ancient pathogen genomes create public-memory pressure because disease headlines
turn sequence data into dramatic plague stories.

| Memory claim | Handling |
|---|---|
| Oldest plague | Treat as dated pathogen evidence, not complete disease history. |
| Pandemic origin | Require phylogeny, sampling, dating, ecology, and historical synthesis custody. |
| Cause of death | Hand to medical systems unless osteological and contextual evidence is locked. |
| Population collapse | Hand to MAXIM/LUCIA unless demographic evidence is locked. |
| Migration by plague | Hand to PORTO/MAXIM unless route evidence is independently locked. |
| Public health analogy | Keep as reception, not archaeological proof. |
