# Claim Audit: Antikythera Reconstruction

**Owner:** ERRATA
**Status:** working audit

## Claim Grades

| Claim | Max grade now | Audit note |
|---|---|---|
| The Antikythera mechanism survives as fragmentary bronze remains. | Observed | Direct material artifact exists, though preservation is damaged and incomplete. |
| Some gears and inscriptions are directly preserved or image-detected. | Corroborated | Imaging and conservation sources must be locked before detailed claims. |
| The device modeled astronomical cycles. | Plausible | Strong scholarly consensus target, but this draft needs source lock for details. |
| A modern reconstruction is the ancient mechanism. | Refuse | Replicas and models are modern objects built from assumptions. |
| The 2021 Cosmos model fully solves the front display. | Speculative | Treat as an important model, not final closure. |
| The mechanism proves a continuous lost industrial tradition. | Speculative | It shows sophisticated technology; broader production ecology requires other evidence. |
| The object is "the first computer" without qualification. | Challenge | Useful public shorthand, but it compresses analog, astronomical, and cultural meanings. |

## Phrases To Challenge

| Phrase | Replacement |
|---|---|
| "the mechanism was exactly..." | "the reconstruction proposes..." |
| "the ancient computer proves..." | "the artifact and models suggest..." |
| "lost technology solved" | "a working reconstruction under specified assumptions" |
| "we know all gears" | "some gears survive; others are inferred" |
| "the Greeks had modern computers" | "a sophisticated analog astronomical device survives in fragmentary form" |

## Source-Locked Claim Updates

|---|---|---|

| The 2021 Cosmos model solves the front display. | Refuse as closure | It is a model that aims to satisfy evidence, not the ancient object itself. |
| NYU collection-level custody permits item-level PTM claims. | Refuse | Collection custody is established; item-level claims require item audit. |
| Current source-locked claims lack page anchors. | Closed | `page-audit.md` anchors current high-level claims. |
