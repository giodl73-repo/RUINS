# Antikythera Reconstruction

**Authors:** draft
**Status:** working draft
**Date:** 2026-07-04

## Abstract

The Antikythera mechanism is a strong RUINS case because the surviving object is
real, but much of its operation is known through imaging, inscription reading,
mathematical modeling, and modern reconstruction. This paper treats the device
as a damaged technical artifact whose partial survival invites overconfidence:
visible gears are not the same thing as hidden scan evidence, scan evidence is
not the same thing as a complete gear train, and a working replica is not the
ancient object. RUINS separates material survival, analytic recovery,
reconstruction, and public technological memory.

## 1. Why This Is A RUINS Case

The Antikythera mechanism is not merely a puzzle object. It is a boundary case
between artifact and reconstruction:

- fragments survive,
- some gears and inscriptions survive directly,
- other features are read through imaging,
- gear trains are modeled mathematically,
- replicas test possible operation,
- public memory turns the device into an "ancient computer" emblem.

The loss is not total. The danger is treating a reconstruction as if it were the
same epistemic object as the surviving fragments.

## 2. What Survives

| Surviving trace | Evidence condition | Claim grade |
|---|---|---|
| Bronze fragments | direct material evidence | observed |
| Visible gears and plates | direct material evidence through conservation | observed / corroborated |
| Inscriptions on fragments | direct material plus imaging/reading | corroborated with source lock |
| Hidden internal features revealed by CT | analytic evidence surface | plausible / source-dependent |
| Gear-train diagrams | reconstruction | model |
| Working replicas | modern reconstruction | test object, not original |

The important discipline is that each layer has a different authority. A gear
tooth preserved in bronze, a tooth count inferred from imaging, and a gear
added to make a model work are not the same kind of evidence.

## 3. What Is Missing

| Missing or weak bridge | Why it matters |
|---|---|
| Complete original device | Prevents direct observation of the whole mechanism. |
| All gears and axes | Makes some gear-train elements reconstructed. |
| Full inscriptional surface | Limits direct reading of intended use and display labels. |
| Workshop context | Prevents confident claims about production tradition. |
| User documentation beyond surviving inscriptions | Limits certainty about operation and audience. |

## 4. Claim Grades

| Claim | Grade | Reason |
|---|---|---|
| The object is a fragmentary ancient geared mechanism. | Observed | Surviving fragments and gears exist. |
| The mechanism modeled astronomical cycles. | Plausible | Major scholarship supports this, pending source lock for detailed wording. |
| CT imaging changed what could be read. | Plausible | Imaging is a core evidence surface, pending source lock. |
| The full front display is solved. | Speculative | Current models are reconstructions under assumptions. |
| A working replica is the ancient mechanism. | Refuse | A replica is a modern model. |
| The object proves a broad lost industrial tradition. | Speculative | Broader production context requires independent evidence. |

## 5. Reconstruction Status

| Reconstruction type | Status | RUINS treatment |
|---|---|---|
| Preserved fragment layout | Direct artifact / conservation record | Use with source custody. |
| Hidden gear reading | Imaging-derived evidence | Use only with imaging source lock. |
| Back-dial function | Strong reconstruction target | Source-lock before detail claims. |
| Front cosmos display | Current model target | Treat as model, not closure. |
| Physical replicas | Experimental reconstruction | Useful as feasibility tests. |
| "Ancient computer" label | Public shorthand | Cultural memory; define carefully. |

## 6. Cultural Memory

| Memory form | What it does | RUINS handling |
|---|---|---|
| Ancient computer framing | Makes the device legible to modern audiences. | Useful but imprecise shorthand. |
| Lost technology narrative | Suggests a vanished technical tradition. | Treat as hypothesis needing broader evidence. |
| Genius-object framing | Turns a damaged artifact into a singular miracle. | Preserve craft and source uncertainty. |
| Replica spectacle | Makes reconstruction visually convincing. | Separate modern build from ancient evidence. |

## 7. Refusals

RUINS refuses the following claims in this foundation draft:

- It will not treat a replica as the original mechanism.
- It will not treat a model as final because it works mechanically.
- It will not merge visible, scan-detected, and inferred components.
- It will not claim a continuous lost industrial tradition from one artifact.
- It will not use "ancient computer" without explaining the analog and
  astronomical context.

## 9. Conclusion

The Antikythera mechanism is a RUINS case because it survives enough to support
technical reconstruction and withholds enough to make reconstruction dangerous.
Responsible handling means keeping bronze, scan, diagram, replica, and public
myth in separate evidence layers.
