# Archaeobotanical Macroremain Custody

**Track:** Evidence and Absence
**Status:** publication-ready
**Rubric:** v1.62
**Score:** 46/50
**Capability decision:** promote `ruins-capability:archaeobotanical-macroremain-custody` to Core

This package uses archaeobotanical macroremain, charred seed, crop-processing,
flotation, and taphonomy studies as the trigger case for macroremain custody. It
keeps samples, recovery, preservation, contexts, taxa, crop/weed status,
taphonomy, processing stage, quantification, spatial distribution, seasonality,
activity inference, economy synthesis, and public plant-food memory separate.
