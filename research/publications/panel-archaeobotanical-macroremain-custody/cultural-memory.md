# Cultural Memory

Archaeobotanical macroremains create public-memory pressure because seeds and
grains invite direct ancient-food and farming stories.

| Memory claim | Handling |
|---|---|
| First farming | Require domestication, chronology, ecology, and regional synthesis custody. |
| Ancient meal | Treat as public story unless cooking, context, ingredients, and use are locked. |
| Granary | Require storage architecture, deposit formation, crop mix, and chronology. |
| Famine | Hand to MAXIM/LUCIA unless scarcity, demography, texts, and context are locked. |
| Ritual deposit | Hand to RITE unless deposition and meaning are independently locked. |
| Market supply | Hand to economy systems; counts alone are not market evidence. |
