# Archaeometallurgical Provenance Custody

**Track:** Evidence and Absence
**Status:** publication-ready
**Rubric:** v1.61
**Score:** 46/50
**Capability decision:** promote `ruins-capability:archaeometallurgical-provenance-custody` to Core

This package uses lead-isotope provenance, slag-inclusion, slag-characterization,
and metal-composition studies as the trigger case for archaeometallurgical
provenance custody. It keeps samples, contexts, analytical methods, corrosion,
ore fields, isotope overlap, reference databases, recycling/mixing, production
stages, provenance probability, exchange inference, and public industry memory
separate.
