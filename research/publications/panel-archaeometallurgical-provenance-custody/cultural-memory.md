# Cultural Memory

Archaeometallurgical provenance creates public-memory pressure because lab
signatures can look like direct maps of ancient industry.

| Memory claim | Handling |
|---|---|
| Ancient trade network | Treat as synthesis unless exchange evidence is separately locked. |
| Exact mine origin | Require isotope overlap, database, geology, artefact, and context custody. |
| Master workshop | Refuse unless workshop debris, production sequence, and object chain are locked. |
| Technological superiority | Treat as public-history rhetoric without comparative production custody. |
| Ethnic or craft identity | Hand to identity systems; chemistry is not identity proof. |
| Solved metal economy | Hand to MAXIM/LUCIA and economy systems. |
