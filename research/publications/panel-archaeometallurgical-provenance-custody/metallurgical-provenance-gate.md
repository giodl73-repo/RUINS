# Metallurgical Provenance Gate

| Gate | Required question |
|---|---|
| Sample custody | Which artefact, slag, inclusion, ore, furnace, crucible, tuyere, prill, or corrosion layer was sampled? |
| Context custody | Is the sample primary production, smithing, repair, recycling, use, loss, discard, or later disturbance? |
| Method custody | Which method, calibration, preparation, detection limit, and uncertainty produced the signature? |
| Alteration custody | Are corrosion, burial chemistry, conservation treatment, contamination, and sampling damage separated? |
| Ore-field custody | Which possible ore fields, overlaps, and database absences are named? |
| Mixing custody | Are alloying, recycling, remelting, and multi-source blending tested before source claims? |
| Production-stage custody | Is the claim ore extraction, smelting, refining, smithing, casting, repair, or exchange? |
| Inference custody | Is the result exact, probable, compatible, ambiguous, or refused? |
| Memory custody | Are ancient-industry, trade-network, superiority, and origin stories contained? |
