## Memory Layers

| Memory layer | Evidence | RUINS treatment |
|---|---|---|
| Lost genius | Archimedes as emblem of ancient science recovered from erasure. | Cultural afterlife, not text completeness. |
| Eureka story | Public romance around rediscovery and scientific brilliance. | Reception layer. |
| Technology rescue | X-rays, multispectral imaging, and processing as heroic recovery. | Method afterlife, not automatic truth. |
| Prayer book tension | Later liturgical use framed as obstacle to ancient text. | The prayer book remains the surviving object. |
| Open data | Raw imaging data and transcriptions published online. | Access layer, not final scholarly closure. |
| Forgery drama | Later paintings obscure and sensationalize the manuscript. | Damage and market-memory layer. |

## Reader Rule

Palimpsest recovery stories should name each layer before making a claim:
parchment, undertext, overtext, damage, conservation, image, processing,
reading, edition, and memory.
