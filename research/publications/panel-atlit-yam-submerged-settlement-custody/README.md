# Atlit-Yam Submerged Settlement Custody

Status: publication-ready
Track: Evidence and Absence
Rubric: v1.16
Score: 46/50

Atlit-Yam is the in-situ submerged-settlement test for Doggerland's broader
proxy gate. It preserves underwater excavation evidence for a Neolithic coastal
settlement: stone-lined wells, rectangular buildings, human graves, megalithic
features, tools, animal and plant remains, and sea-level process.
