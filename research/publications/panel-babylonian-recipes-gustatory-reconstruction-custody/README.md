# Babylonian Recipes Gustatory Reconstruction Custody

**Track:** Evidence and Absence
**Status:** publication-ready
**Rubric:** v1.30
**Score:** 46/50
**Capability decision:** promote `ruins-capability:gustatory-reconstruction-custody` to Core

This package treats the Yale Babylonian cooking tablets as a RUINS case about
text-driven taste reconstruction. It separates tablet witness, Akkadian recipe
language, translation, missing measures, ingredient substitution, experimental
cooking, public recipes, and oldest-recipe memory.
