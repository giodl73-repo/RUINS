# Gustatory Reconstruction Gate

| Layer | Babylonian recipe custody question |
|---|---|
| Tablet | Does the claim stay anchored to the clay recipe witness? |
| Text | Does it separate Akkadian text, transliteration, translation, and recipe paraphrase? |
| Damage | Are missing or terse instructions treated as evidence limits? |
| Ingredient | Are ancient terms and modern substitutions kept separate? |
| Process | Are quantities, timing, heat, vessels, and technique future-locked? |
| Experiment | Is the cooked dish labeled as reconstruction or teaching tool? |
| Public recipe | Is modern recipe formatting treated as access layer? |
| Safety | Are cooking and food-safety claims excluded? |
| Memory | Does "oldest recipe" stay public memory rather than exact taste proof? |
