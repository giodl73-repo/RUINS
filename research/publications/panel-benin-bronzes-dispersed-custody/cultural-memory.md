**Owner:** Cultural Memory Reader

## Memory Forms

| Memory form | Evidence surface | RUINS treatment |
|---|---|---|
| Restitution as repair | Public return narratives | Afterlife and ethics layer; not automatic total context restoration. |
| Universal museum story | Museum public framing | Institutional memory and display logic. |
| Cultural ambassadors | International display language | Reception layer; not ownership settlement. |
| Digital reconnection | Digital Benin and museum metadata | Access/custody layer, not object equivalent. |
| Oral history | Edo knowledge transmission | Living-memory surface requiring community-source care. |
| Looted treasure image | Public colonial-violence memory | Valid memory layer, not object-level provenance by itself. |

## Publication Rules

1. Separate return, repair, access, display, and ownership language.
2. Treat oral history as living custody, not decorative supplement.
3. Treat digital reconnection as a knowledge layer, not restitution.
4. Avoid totalizing the corpus from one museum's return or retention position.
