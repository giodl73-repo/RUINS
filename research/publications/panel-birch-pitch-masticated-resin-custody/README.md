# Birch Pitch Masticated Resin Custody

**Track:** Evidence and Absence
**Status:** publication-ready
**Rubric:** v1.39
**Score:** 46/50
**Capability decision:** promote `ruins-capability:masticated-resin-biomolecular-custody` to Core

This package uses the 5,700-year-old chewed birch pitch from Syltholm, Denmark,
as the trigger case for masticated-resin custody. It keeps object identity,
chewing evidence, ancient genome, oral microbiome, food DNA, microbial/pathogen
signals, adhesive or medicinal pitch use, and public ancient-chewing-gum memory
separate.
