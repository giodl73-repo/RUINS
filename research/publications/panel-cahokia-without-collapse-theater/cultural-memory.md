# Cultural Memory Register: Cahokia Without Collapse Theater

**Owner:** Cultural Memory Reader
**Status:** first register
**Date:** 2026-07-04

## Memory Forms

| Memory form | Evidence surface | RUINS treatment |
|---|---|---|
| "Lost civilization" | Public and popular framing; explicitly challenged by Berkeley's summary of White et al. | Treat as modern narrative form, not as an archaeological conclusion. |
| "Vanishing Indian" pattern | Named in the Berkeley summary as a narrative that favors disappearance over resilience and persistence. | Refuse disappearance language unless a source makes a narrow demographic claim. |
| "Collapse" theater | Appears in scholarship and public summaries as a shorthand for decline/abandonment. | Grade each use; prefer process words unless source-specific. |
| "America's first city" / major urban antiquity | Public heritage and educational framing that counters assumptions about Indigenous urbanism. | Useful cultural-memory framing; not Cahokian self-description. |
| Ecocide lesson | Wood-overuse and environmental-mismanagement narratives reused as modern warnings. | Treat as modern analogy requiring stratigraphic and proxy support. |
| World Heritage site | UNESCO and state-site framing. | Custody and public afterlife evidence. |

## Publication Rules

1. Keep "lost," "vanished," and "mysterious" words in quotation or critique
   context unless source-locked as public-memory language.
2. Use "collapse" only when discussing a source that uses collapse vocabulary
   or when describing the narrative this paper refuses.
3. Do not turn Indigenous continuity into a generic reassurance. Name the
   source, scope, and evidence type.
4. Separate heritage recognition from archaeological proof.
