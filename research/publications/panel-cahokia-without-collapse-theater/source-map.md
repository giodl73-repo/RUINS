**Owner:** PALIMPSEST
**Status:** working source map
**Rule:** metadata-only; do not copy protected source content

## Evidence Surfaces

| Surface | Evidence condition | RUINS use |
|---|---|---|
| Surviving mounds and earthworks | direct material landscape evidence | Establish surviving urban/ceremonial scale. |
| Excavated features and artifacts | direct material evidence mediated by excavation records | Track households, plazas, burials, craft, foodways, and social organization. |
| Sediment, flood, and paleoenvironmental records | proxy evidence | Evaluate environmental hypotheses without overclaiming causation. |
| Population and demographic models | reconstruction / analytic model | Frame peak and decline cautiously. |
| Oral history / descendant community context | living memory and ethical context | Prevent "vanished civilization" erasure. |
| Public site interpretation and media | cultural memory | Track lost-city, mystery, and collapse narratives. |

## Source Targets

| Target | Use | Claim boundary |
|---|---|---|
| Cahokia Mounds State Historic Site / UNESCO World Heritage materials | Site custody, public heritage status, broad mound/site facts. | Public-site materials are not enough for detailed archaeological claims. |
| PNAS flood-frequency paper on Cahokia emergence/decline | Environmental proxy hypothesis. | Flood correlation is not single-cause proof. |
| WashU 2021 wood-overuse / flooding critique | Tests deforestation-flood-collapse model. | Negative evidence for one hypothesis, not a complete explanation. |
| Berkeley 2020 demographic/continuity framing | Challenges lost-civilization abandonment narrative. | Public university summary; source-lock before detailed use. |
| Frontiers urbanization/metabolism paper | Collapse/societal metabolism model. | Interpretive model, not settled cause. |
| Mound 72 and Monks Mound literature | Feature-level evidence targets. | Object-level or burial claims require page audit and sensitivity review. |
