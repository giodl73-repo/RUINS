# Canopy LiDAR Settlement Custody

**Track:** Evidence and Absence
**Status:** publication-ready
**Rubric:** v1.59
**Score:** 46/50
**Capability decision:** promote `ruins-capability:canopy-lidar-settlement-custody` to Core

This package uses airborne LiDAR papers from Maya, Angkor, Amazonian, and
wetland-field archaeology as the trigger case for canopy-settlement custody. It
keeps sensor flights, point clouds, return classification, canopy filtering,
terrain models, visualizations, feature candidates, ground truthing,
chronology, functional interpretation, population models, landscape boundaries,
publication images, and public lost-city memory separate.
