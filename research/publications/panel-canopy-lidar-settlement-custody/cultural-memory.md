# Cultural Memory

LiDAR discovery papers create public-memory pressure because visualizations make
hidden terrain look like solved history.

| Memory claim | Handling |
|---|---|
| Lost city | Treat as public discovery story unless a named site and evidence chain are separately locked. |
| Rediscovered civilization | Refuse as totalizing language; keep to bounded feature and landscape claims. |
| Jungle revealed | Keep as visualization rhetoric, not proof that survey is complete. |
| Megacity | Require population, chronology, occupation, and settlement-definition custody. |
| Collapse solved | Hand to MAXIM/LUCIA and climate/history systems; LiDAR alone does not solve causation. |
| Conservation-complete map | Requires separate governance, field verification, threat, and access evidence. |
