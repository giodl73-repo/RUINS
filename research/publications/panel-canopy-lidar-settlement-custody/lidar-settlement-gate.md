# LiDAR Settlement Gate

| Gate | Required question |
|---|---|
| Flight custody | Which airborne survey, instrument, date, altitude, coverage polygon, and point density produced the evidence? |
| Point classification | Which returns are ground, vegetation, building-like, noise, or unclassified? |
| Canopy filtering | What filtering choices convert dense vegetation into a bare-earth model? |
| Visualization | Is the claim from raw point cloud, DTM, hillshade, slope, sky-view factor, or composite interpretive image? |
| Candidate status | Is the mapped object a candidate feature or a ground-confirmed archaeological feature? |
| Chronology | Is the feature dated independently, or only visible in the terrain model? |
| Function | Are settlement, agricultural, hydraulic, ritual, defensive, and natural explanations separately evaluated? |
| Scale model | Are survey polygon, extrapolation, omission/commission error, and population assumptions explicit? |
| Memory | Are lost-city, rediscovered-civilization, and collapse stories held as public-memory claims? |
