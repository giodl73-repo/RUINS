# Catalhoyuk Partial Excavation Shelter Custody

Status: publication-ready
Track: Evidence and Absence
Rubric: v1.21
Score: 46/50

Catalhoyuk is a RUINS second test for partial excavation shelter custody. Unlike
Gobekli Tepe's monumental stone exposure, Catalhoyuk is a mudbrick settlement
where shelters, visitor routes, signs, replica objects, off-site collections,
archives, and ongoing conservation all mediate what the public can responsibly
know.
