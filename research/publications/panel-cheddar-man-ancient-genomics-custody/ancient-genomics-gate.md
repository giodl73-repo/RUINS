# Ancient Genomics Gate

| Layer | Custody question |
|---|---|
| Sample | Which bone, tooth, extract, library, or genome dataset anchors the claim? |
| Authentication | What contamination, coverage, damage pattern, and quality checks constrain use? |
| Reference panel | Which ancient and modern populations are used for comparison? |
| Model | Is the claim PCA, ADMIXTURE, qpAdm, kinship, phenotype prediction, or another model? |
| Phenotype | Which marker supports eye, hair, skin, lactase, or trait inference, and at what certainty? |
| Reconstruction | What facial, artistic, or media choices are added beyond DNA? |
| Population | Is the claim individual, group affinity, replacement, continuity, or admixture? |
| Identity | Are modern race, ethnicity, nation, and direct ancestry separately refused or sourced? |
| Memory | Is public "first Briton" or origin language separated from genomic evidence? |
