# Cultural Memory Register: Dead Sea Scrolls Fragment Custody

**Owner:** Cultural Memory Reader
**Status:** working register

## Memory Surfaces

| Memory surface | What it preserves | Risk |
|---|---|---|
| Scriptural witness | The fragments matter to communities and textual traditions. | Religious importance can be mistaken for stronger material certainty. |
| Hidden-scroll secrecy | Publication delay and access limits become conspiracy narrative. | Public suspicion can become false evidence. |
| Archaeological treasure | Discovery drama dominates custody. | Treasure framing can hide conservation and provenance. |
| Museum display | Fragments become heritage and education objects. | Display label can be mistaken for full source audit. |
| Forgery scandal | Modern authenticity disputes become public drama. | One fragment set can be generalized to the whole corpus. |

## RUINS Treatment

Cultural memory around the scrolls is part of the evidence history. RUINS should
not erase scripture, secrecy, heritage, or authenticity drama, but it must keep
those afterlives separate from physical fragment custody.
