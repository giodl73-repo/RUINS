# Named Fragment Anchor Audit: Dead Sea Scrolls Fragment Custody

**Owner:** PALIMPSEST / ERRATA
**Status:** current-scope multi-example image/metadata audit
**Date:** 2026-07-05

This audit locks named image/metadata examples for current RUINS use. It does
not authorize exact readings, transcriptions, translations, joins, lacunae, or
authenticity conclusions.

## Selected Examples

The first audited example is the IAA / Leon Levy archive record for images of
11Q5 / 11Q Psalms. The second is an IAA / Leon Levy archive record for 4Q7 /
4Q Genesis. A third multi-manuscript image record is included as a boundary
example because it shows that one image can carry multiple manuscript labels.
RUINS uses these examples because they expose the custody surfaces this paper
must keep separate: image id, plate, date, photographer, image type, side,
manuscript label, and fragment relation.

## Current Decision

The named-fragment image/metadata blocker is closed for current
publication-ready custody-ladder claims. RUINS may use the 11Q5 / 11Q Psalms
and 4Q7 / 4Q Genesis records as examples of mediated image custody and public
metadata structure, and may use B-298939 as a boundary example for
multi-manuscript image metadata. Exact readings, edition pages, joins, lacunae,
and translation examples remain outside current scope.
