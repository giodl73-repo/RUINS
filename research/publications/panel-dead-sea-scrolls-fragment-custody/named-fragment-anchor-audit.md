# Named Fragment Anchor Audit: Dead Sea Scrolls Fragment Custody

**Owner:** PALIMPSEST / ERRATA
**Status:** current-scope image/metadata audit
**Date:** 2026-07-05

This audit locks a first named-fragment image/metadata example for current
RUINS use. It does not authorize exact readings, transcriptions, translations,
joins, lacunae, or authenticity conclusions.

## Selected Example

The first audited example is the IAA / Leon Levy archive record for images of
11Q5 / 11Q Psalms. RUINS uses it because it exposes the custody surfaces this
paper needs to keep separate: image id, plate, date, photographer, image type,
side, manuscript label, and fragment relation.

## Current Decision

The named-fragment image/metadata blocker is closed for current stable claims.
RUINS may now use 11Q5 / 11Q Psalms image records as an example of mediated
image custody and public metadata structure. The paper remains capped at stable
working paper because exact readings, edition pages, translation examples, and
direct authenticity-report custody remain unaudited.
