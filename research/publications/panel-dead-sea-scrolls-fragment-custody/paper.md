# Dead Sea Scrolls Fragment Custody

**Authors:** draft
**Status:** stable working paper
**Date:** 2026-07-05

## Abstract

The Dead Sea Scrolls are a RUINS case because the surviving evidence is both
abundant and fragmentary. Thousands of fragments, photographs, editions,
translations, reconstructed columns, collection histories, museum displays, and
authenticity disputes make the corpus visible but uneven. This paper treats the
scrolls as a test of fragment custody: how to say what survives without letting
editions, translations, reconstructions, or public memory masquerade as the
fragment itself.

## 1. Why This Is A RUINS Case

The Dead Sea Scrolls are not a ruin because they disappeared. They are a ruin
because survival is granular. A letter may survive while the line around it is
reconstructed. A fragment may be photographed, edited, translated, joined,
renumbered, displayed, disputed, or reclassified. Each surface is useful, but
each surface supports different claims.

This case asks:

- what the physical fragment can establish,
- when a photograph or edition carries custody,
- how translation should be separated from surviving text,
- how reconstructed gaps should be marked,
- how provenance and authenticity affect claim grade,
- how public memory around scripture and secrecy changes reception.

## 2. What Survives

| Surviving trace | Evidence condition | Claim grade now |
|---|---|---|
| Physical fragments from cave and collection contexts | direct material | source-specific |
| Plates, photographs, and digital images | mediated material record | source-specific |
| Scholarly editions and transcriptions | edited textual custody | source-specific |
| Translations | interpretation / translation | source-specific |
| Reconstructed columns, joins, and lacunae | scholarly reconstruction | plausible or speculative |
| Provenance records and collection histories | custody / reported history | source-specific |
| Authenticity reports for modern-market fragments | material analysis / dispute record | source-specific |
| Museum labels, documentaries, and public claims | cultural memory | afterlife evidence |

## 3. What Is Missing Or Damaged

| Missing or weak bridge | Why it matters |
|---|---|
| Complete manuscripts | Many witnesses survive as fragments, not whole scrolls. |
| Undisturbed discovery context for every fragment | Acquisition and dispersal histories affect custody. |
| Single stable numbering/identity surface | Cave, manuscript, fragment, edition, and collection labels can diverge. |
| Clear boundary between surviving text and editorial completion | Readers can mistake reconstruction for preserved wording. |
| Uniform authenticity status across modern-market fragments | Some fragments require separate material/authenticity treatment. |

## 4. Claim Grades

| Claim | Grade | Reason |
|---|---|---|
| Dead Sea Scrolls fragments survive materially. | Observed / source-specific | Direct fragments exist, but exact fragment claims need source custody. |
| A photograph or digital image preserves a useful witness. | Source-specific | Image custody is mediated and must name source/plate. |
| 11Q5 / 11Q Psalms image records can illustrate image custody. | Source-specific | Named IAA/Leon Levy records anchor image id, plate, image type, side, manuscript label, and fragment relation only. |
| A transcription or edition equals the fragment. | Refuse | Edition is scholarly mediation. |
| A translation equals surviving wording. | Refuse | Translation is interpretation. |
| A reconstructed lacuna is ancient text. | Refuse | Reconstruction is not survival unless marked and constrained. |
| A fragment's cave provenance is settled by a collection label. | Speculative / source-specific | Provenance needs custody, not assumption. |
| A modern authenticity dispute invalidates the entire corpus. | Refuse | Authenticity concerns are fragment/set-specific. |

## 5. Reconstruction Status

| Reconstruction type | Status | RUINS treatment |
|---|---|---|
| Fragment join | Scholarly hypothesis | Use only with source and confidence. |
| Lacuna completion | Modern reconstruction | Mark as supplied, not surviving. |
| Column/order reconstruction | Scholarly model | Useful but not equivalent to intact manuscript. |
| Translation | Interpretation | Source-specific and language-dependent. |
| Museum display narrative | Public image | Cultural memory and education surface. |
| Forgery determination | Material/authenticity claim | Fragment-specific source lock required. |

## 6. Cultural Memory

| Memory form | What it does | RUINS handling |
|---|---|---|
| Biblical/scriptural authority | Makes fragments feel like direct access to origins. | Keep religious importance separate from evidence grade. |
| Secrecy/conspiracy story | Turns publication delay into hidden-knowledge narrative. | Treat as public memory unless sourced as publication history. |
| Archaeological treasure image | Makes discovery more important than custody. | Refuse treasure framing. |
| Forgery scandal | Makes authenticity drama dominate the corpus. | Keep fragment-specific. |
| National/museum heritage | Places fragments in modern custody and identity. | Afterlife evidence, not original-state proof. |

## 7. Refusals

RUINS refuses the following claims in this stable working paper:

- It will not treat translation as the fragment.
- It will not treat reconstructed lacunae as surviving wording.
- It will not infer provenance from display label alone.
- It will not generalize a forgery finding from one fragment set to the whole
  corpus.
- It will not turn scriptural importance into stronger material evidence.
- It will not make secrecy, treasure, or sensational authenticity claims without
  source custody.

## 9. Conclusion

The Dead Sea Scrolls are a strong RUINS case because they preserve enough text to
invite direct contact and enough fracture to punish careless certainty. A
responsible account must keep fragment, photograph, edition, translation,
reconstruction, provenance, authenticity, and cultural memory in separate
evidence layers.
