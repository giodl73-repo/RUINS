# Publication Scope Gate: Dead Sea Scrolls Fragment Custody

**Owner:** ASHLAR / PALIMPSEST / ERRATA
**Status:** current-scope gate closed
**Date:** 2026-07-05

This gate closes publication-ready scope for the current paper by separating
custody-ladder claims from exact textual claims. The paper is publication-ready
as a non-reading RUINS methodology case: it may explain how fragment, image,
metadata, edition, translation, reconstruction, provenance, authenticity, and
cultural memory differ, but it may not publish exact readings or reconstruction
claims from named fragments.

## Current Scope

| Claim surface | Current RUINS permission |
|---|---|
| Fragmentary survival of the Dead Sea Scrolls corpus | Allowed, source-scoped to official public custody. |
| Digital image and metadata custody | Allowed, source-scoped to IAA / Leon Levy public image and metadata pages. |
| Named image/metadata examples such as 11Q5, 4Q7, and B-298939 | Allowed as image/metadata examples only. |
| Edition target identification | Allowed as bibliographic target custody. |
| MOTB authenticity finding | Allowed only for the tested MOTB 16-fragment set. |
| Translation, reconstruction, and lacuna caution | Allowed as RUINS method boundaries. |
| Exact ancient wording, transcription, translation, join, lacuna, or textual variant | Refuse until direct edition-page custody is added. |
| Whole-corpus forgery conclusion | Refuse. |
| Identifier certainty across cave, manuscript, fragment, edition, and collection systems | Refuse without source-specific identity custody. |

## Publication Rule

Publication-ready status here means the paper is ready as a custody-ladder
argument. It does not mean the paper has edition-level authority to quote,
transcribe, translate, join, reconstruct, or adjudicate textual variants.

## Future Exact-Claim Minimum

Before RUINS adds exact textual claims, it must record:

1. Edition or lawful page-audited source.
2. Page, column, line, plate, and image relation.
3. Whether text is visible, supplied, reconstructed, translated, or joined.
4. Identifier system used: cave, manuscript, fragment, edition, collection, or
   image id.
5. Claim grade and sibling handoff if the claim belongs to FONTES, MUNDUS, or
   CANON rather than RUINS.

## Decision

No open P2 remains for the current paper. Edition-page custody remains a P3
future-use gate for exact textual claims, not a current blocker for the
non-reading publication scope.
