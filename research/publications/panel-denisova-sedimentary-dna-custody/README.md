# Denisova Sedimentary DNA Custody

**Track:** Evidence and Absence
**Status:** publication-ready
**Rubric:** v1.45
**Score:** 46/50
**Capability decision:** promote `ruins-capability:sedimentary-ancient-dna-custody` to Core

This package uses Denisova Cave sedimentary ancient DNA as the trigger case for
sedimentary-DNA custody. It keeps sediment samples, grid collection,
stratigraphy, microstratigraphic hotspots, mtDNA/nuclear DNA, faunal DNA,
hominin lineage assignment, artefact association, occupation history, absence
claims, and public no-bones-needed memory separate.
