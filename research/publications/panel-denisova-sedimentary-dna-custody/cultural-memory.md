# Cultural Memory Register

| Memory surface | RUINS handling |
|---|---|
| No bones needed | Public method memory, not complete evidence replacement. |
| Cave occupants | Presence shorthand requiring sample and time custody. |
| Neanderthals met Denisovans | Encounter memory, not co-occurrence proof. |
| DNA writes occupation history | Synthesis claim requiring stratigraphy and association. |
| Artefact makers identified | Refused unless artefact linkage is direct. |
| Missing DNA means nobody was there | Absence memory requiring detection custody. |
