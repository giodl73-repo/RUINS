# Doggerland Submerged Landscape Proxy

Status: publication-ready
Track: Evidence and Absence
Rubric: v1.15
Score: 45/50

Doggerland is a RUINS case for submerged landscapes known through indirect and
uneven survival: seismic survey, bathymetry, sediment cores, trawled artefacts,
faunal remains, ancient environmental DNA, inundation modeling, and public
"lost world" memory.
