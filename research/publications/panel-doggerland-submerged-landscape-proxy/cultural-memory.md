# Cultural Memory: Doggerland

Doggerland's public afterlife often centers on "lost world" language. RUINS
keeps that language as reception evidence, not as original-state proof.

## Memory Surfaces

| Memory surface | RUINS handling |
|---|---|
| Lost world framing | Evidence of public imagination and science communication. |
| Atlantis analogy | Refused as evidence; useful only as a warning sign for overclaim. |
| Climate parable | May explain modern relevance, not Mesolithic lived experience. |
| Full-color reconstruction maps | Modern visualization requiring model/source custody. |
| Disaster narrative | Refused unless tied to a specific event source and scope. |

## Rule

Public memory may explain why Doggerland matters now. It may not fill missing
settlement, identity, route, or event evidence.
