**Track:** Evidence and Absence

This package treats Dresden's Frauenkirche as a RUINS case about monument-scale
archival reconstruction. It separates rubble, reused stones, historical plans,
photogrammetry, structural redesign, new sandstone, reconstructed interior work,
and peace/reconciliation memory.
