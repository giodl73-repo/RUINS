# Archival Reconstruction Gate

| Layer | Dresden custody question |
|---|---|
| Loss | Does the claim name wartime destruction and postwar rubble without turning that into a complete loss inventory? |
| Surviving fabric | Does it separate historical building fabric from new sandstone and modern structure? |
| Find custody | Does it distinguish mapped, recovered, and catalogued finds from reused fabric claims? |
| Archive | Does it specify historical plans, floor plans, and reconstruction documentation rather than generic memory? |
| Model | Does it label photogrammetry and computer-aided 3D modeling as mediation? |
| Reconstruction | Does it separate historically faithful aim from modern structural corrections and material substitutions? |
| Interior | Does it keep artistic interior work and organ/altar claims future-locked unless separately sourced? |
| Memory | Does it treat peace and reconciliation as afterlife rather than proof of complete recovery? |
