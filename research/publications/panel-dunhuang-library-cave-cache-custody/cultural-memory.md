# Cultural Memory: Dunhuang Library Cave

Cave 17 is often remembered as a secret library, treasure chamber, or digitally
excavated archive. RUINS keeps those phrases as afterlife evidence.

## Memory Surfaces

| Memory surface | RUINS handling |
|---|---|
| "Library Cave" | Useful conventional name; not proof of library function. |
| Secret library | Public-memory phrase, not deposition evidence. |
| Treasure discovery | Reception frame; may obscure custody and dispersal. |
| Earliest printed book focus | Important item-level memory, not whole-cache interpretation. |
| Digital excavation/reunion | Access and comparison claim, not physical reunification. |

## Rule

Public fascination may explain Cave 17's modern importance. It may not supply
original order, function, motive, or representativeness.
