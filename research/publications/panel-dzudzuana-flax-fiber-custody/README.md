# Dzudzuana Flax Fiber Custody

**Track:** Evidence and Absence
**Status:** publication-ready
**Rubric:** v1.44
**Score:** 46/50
**Capability decision:** promote `ruins-capability:perishable-fiber-microtrace-custody` to Core

This package uses 30,000-year-old wild flax fibers from Dzudzuana Cave as the
trigger case for perishable-fiber microtrace custody. It keeps clay samples,
stratigraphy, fiber identity, twist, cut ends, color, possible dyeing, cordage,
basketry, sewing, skin/fur work, textile inference, and oldest-clothing memory
separate.
