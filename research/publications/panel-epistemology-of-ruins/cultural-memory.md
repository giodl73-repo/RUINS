# Cultural Memory Register: The Epistemology of Ruins

**Owner:** Cultural Memory Reader
**Status:** method-level register

Cultural memory is not contamination to be stripped away. It is a later evidence
layer that must be separated from original-state claims.

## Memory Forms

| Memory form | Evidence value | Risk |
|---|---|---|
| Legend | Shows reception, desire, legitimacy, or symbolic reuse. | Can be mistaken for source evidence. |
| National symbol | Shows modern identity and custody. | Can be projected backward into ancient identity. |
| Public image | Shows what a site or object became in popular imagination. | Can replace the evidence surface. |
| Restoration image | Shows modern intervention and visual authority. | Can be mistaken for ancient appearance. |
| Lost-city narrative | Shows modern appetite for closure and spectacle. | Can erase living communities and known evidence. |
| Technical-loss myth | Shows modern desire for secret recovery. | Can hide partial continuity and experimental limits. |

## Case Memory Tests

| Case | Memory pressure | RUINS handling |
|---|---|---|
| Alexandria | One dramatic destruction story. | Separate institutions, episodes, sources, and later fame. |
| Indus | Decipherment desire and civilization identity. | Treat proposed readings as claims, not closure. |
| Knossos | Restored palace image. | Treat restoration as modern afterlife. |
| Great Zimbabwe | Non-African origin myths and national symbolism. | Refuse denial as archaeology; preserve as afterlife evidence. |
| Cahokia | Vanished-people and collapse theater. | Treat abandonment as transformation, not disappearance. |
| Roman concrete | Lost recipe myth. | Separate material survival, textual trace, continuity, and experiment. |
