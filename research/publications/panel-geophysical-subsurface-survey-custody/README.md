# Geophysical Subsurface Survey Custody

**Track:** Evidence and Absence
**Status:** publication-ready
**Rubric:** v1.60
**Score:** 46/50
**Capability decision:** promote `ruins-capability:geophysical-subsurface-survey-custody` to Core

This package uses Falerii Novi GPR, Stonehenge Hidden Landscapes geophysics, and
multi-method archaeological prospection as the trigger case for subsurface
survey custody. It keeps instruments, grids, sampling intervals, processing,
georeferencing, depth models, soil contrast, anomaly classes, interference,
feature candidates, ground truth, chronology, function, plan synthesis, and
public nondestructive-discovery memory separate.
