# Cultural Memory

Geophysical survey creates public-memory pressure because buried features can be
made visible while still unexcavated.

| Memory claim | Handling |
|---|---|
| City without digging | Treat as nondestructive-discovery rhetoric, not excavation-equivalent proof. |
| Hidden monument | Require anomaly class, ground truth, chronology, and function custody. |
| Complete urban plan | Separate survey coverage, interpolation, and untested areas. |
| Sacred landscape | Hand ritual and meaning claims to RITE unless independently locked. |
| Solved site function | Hand broader synthesis to MAXIM/LUCIA. |
| Preservation status | Requires conservation or excavation evidence, not survey image alone. |
