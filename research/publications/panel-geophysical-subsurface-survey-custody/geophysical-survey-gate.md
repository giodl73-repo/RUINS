# Geophysical Survey Gate

| Gate | Required question |
|---|---|
| Instrument custody | Which method, frequency, sensor array, spacing, and acquisition mode produced the signal? |
| Grid custody | Which grid, transects, positioning system, and georeferencing procedure locate the anomaly? |
| Processing custody | Which filters, corrections, interpolations, depth slices, and visualizations shaped the map? |
| Soil contrast | Does the signal depend on moisture, geology, magnetic contrast, stone, burning, fill, metal, or modern debris? |
| Anomaly status | Is the mapped pattern a candidate wall, ditch, pit, pipe, tomb, floor, natural feature, or unknown? |
| Ground truth | Was the candidate tested by excavation, coring, repeat method, legacy plan, or surface evidence? |
| Chronology | Is date independently supported, or only inferred from plan position and analogy? |
| Function | Are urban, domestic, hydraulic, ritual, industrial, defensive, and natural explanations separated? |
| Memory | Are whole-city, hidden-monument, nondestructive certainty, and sacred-landscape stories contained? |
