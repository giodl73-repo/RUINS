# Refusal Register: Great Zimbabwe Colonial Denial

**Owner:** ERRATA / Safety Editor
**Status:** first register
**Date:** 2026-07-04

## Refused Claims

| Claim | Refusal | Allowed substitute |
|---|---|---|
| Great Zimbabwe was built by foreign or non-African visitors. | This is colonial denial, not current archaeology. | Great Zimbabwe was built by local African societies; detailed identity claims remain source-specific. |
| Imported goods prove foreign builders. | Trade/contact evidence does not establish authorship. | Imported goods show long-distance exchange. |
| Great Zimbabwe is an ownerless lost city. | Local memory, sacred use, national symbolism, and heritage custody all contradict ownerless framing. | Great Zimbabwe is a monumental African site with layered custody and memory. |
| The site was abandoned for one simple cause. | Abandonment and transformation require source-specific treatment. | Use source-labeled hypotheses and avoid closure. |
| UNESCO proves every interpretation. | UNESCO is modern heritage recognition. | UNESCO supports public values and custody framing. |
| National symbolism proves medieval identity. | Modern national memory is a later layer. | Treat Zimbabwe Bird/naming as national afterlife. |
| Archaeology should silence local memory. | Living memory is an evidence surface for afterlife and custody. | Hold archaeological, local, and heritage claims apart. |
| Local memory should replace archaeological evidence. | Different claim types require different evidence. | Use local memory for meaning/custody, archaeology for material chronology. |
