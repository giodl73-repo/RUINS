# Greenland Ice-Core Lead Proxy Custody

**Track:** Evidence and Absence
**Status:** publication-ready
**Rubric:** v1.47
**Score:** 46/50
**Capability decision:** promote `ruins-capability:ice-core-pollution-proxy-custody` to Core

This package uses Greenland and Arctic ice-core lead pollution papers as the
trigger case for ice-core pollution proxy custody. It keeps cores, annual
layers, lead concentrations, isotopes, deposition flux, atmospheric transport,
source apportionment, economic inference, historical event alignment,
health-exposure modelling, and public Rome-pollution memory separate.
