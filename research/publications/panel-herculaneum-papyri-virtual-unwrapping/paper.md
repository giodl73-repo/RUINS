# Herculaneum Papyri Virtual Unwrapping

**Authors:** draft
**Status:** publication-ready
**Date:** 2026-07-05

## Abstract

The Herculaneum papyri are a RUINS case because their survival is both material
and computational. The scrolls survived as carbonized objects, but many texts
survived only as unreadable layers until imaging, segmentation, and ink
detection made parts of them legible. This paper treats virtual unwrapping as a
custody ladder: scroll, scan, surface, label, model output, scholarly reading,
and public memory must not be collapsed into one claim.

## 1. Why This Is A RUINS Case

The papyri are not simply "lost texts recovered by AI." They are damaged
objects whose unreadability is part of their evidence condition. Past physical
opening produced some scholarship but also broke and destroyed material. Modern
noninvasive work changes the evidence bridge without removing uncertainty.

This case asks RUINS to separate:

- material survival from textual recovery,
- CT volume custody from visible surface or ink custody,
- segmentation from unwrapping,
- ink detection from transcription,
- recovered text from interpretation,
- public lost-library memory from original-state evidence.

## 2. What Survives

| Surviving trace | Evidence condition | Claim grade now |
|---|---|---|
| Carbonized papyrus scrolls and fragments | direct material | source-specific |
| CT volumes and reconstructed scan data | mediated volumetric record | source-specific |
| Surface segmentations and meshes | computational representation | model-mediated |
| Labels aligned between images and CT | training / validation custody | source-specific |
| Ink detection outputs | model output | plausible / review-dependent |
| Scholarly readings of recovered text | expert interpretation | source-specific |
| Prize announcements and media stories | cultural memory | afterlife evidence |

## 3. Claim Grades

| Claim | Grade | Reason |
|---|---|---|
| The Herculaneum papyri survived the AD 79 eruption in carbonized form. | Source-specific | Public challenge and dataset sources support this broadly. |
| The scrolls can be read directly from raw CT data. | Refuse | Surface representation, geometry, and ink detection remain separate problems. |
| A model output is a transcription. | Refuse | Model output needs scholarly review and text custody. |
| Ink detection proves a complete recovered work. | Refuse | Detection, segmentation, reading, and work identification are different layers. |
| Vesuvius Challenge data can support scan-custody claims. | Source-specific | AWS and project pages expose open data and scan-custody boundaries. |
| "AI recovered a lost library" is original-state evidence. | Refuse | It is public afterlife and recovery narrative. |

## 4. Reconstruction Status

| Layer | RUINS status | Boundary |
|---|---|---|
| Scroll object | material survival | Requires custody and conservation context. |
| CT volume | mediated witness | Not the text itself. |
| Surface segmentation | computational reconstruction | Useful, but not physical unrolling. |
| Ink detection | model inference | Requires validation and scholarly review. |
| Transcription | scholarly text claim | Requires papyrological custody. |
| Translation / work identification | interpretation | Requires source-specific scholarly custody. |

## 5. Cultural Memory

| Memory form | RUINS handling |
|---|---|
| Lost library restored | Afterlife narrative; not proof of recovered contents. |
| AI miracle | Method memory; must be separated from evidence grade. |
| Prize race | Social/research organization evidence, not textual evidence. |
| First-scroll claims | Milestone claims need source and scope. |
| Ancient philosophy recovered | Possible scholarly consequence; not a claim without text custody. |

## 6. Refusals

RUINS refuses the following claims in this stable working paper:

- It will not treat CT volumes as readable text.
- It will not treat segmentation as unrolling without reconstruction status.
- It will not treat model ink detections as transcriptions.
- It will not identify authors, titles, or doctrines without scholarly custody.
- It will not claim that the whole Herculaneum library has been recovered.
- It will not turn prize announcements into textual proof.

## 8. Conclusion

The Herculaneum papyri show that a ruin can become more legible without ceasing
to be a ruin. Virtual unwrapping improves access, but each computational layer
adds its own custody requirement. RUINS should treat this as digitally mediated
partial survival: powerful, promising, and still bounded by source discipline.
