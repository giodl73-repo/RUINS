# Scholarly Text Custody Gate: Herculaneum Papyri Virtual Unwrapping

**Owner:** PALIMPSEST / ERRATA
**Status:** current-scope scholarly custody gate
**Date:** 2026-07-05

This gate defines the current scholarly custody for recovered-text and
work-identity claims. It closes the scholarly-text gate for the paper's current
non-quotation claims. It does not authorize RUINS to reproduce extended Greek
text, publish translations, or make independent philological interpretations.

## Minimum Custody For Future Exact Text

Before RUINS adds any exact recovered-text quotation or translation, it must
record:

1. Scroll/object ID.
2. Dataset/model artifact path, if the claim depends on virtual unwrapping.
3. Papyrological review status.
4. Transcription source.
5. Translation source, if applicable.
6. Copyright/quotation boundary.

## Current Decision

The current paper can be publication-ready because it is methodological: it
does not quote recovered Greek text or make independent interpretive claims.
The scholarly-text gate is closed only for source-scoped, non-quotation claims.
