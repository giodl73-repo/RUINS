# Cultural Memory Register: House of Wisdom Institution Memory

**Owner:** Cultural Memory Reader
**Status:** working register

## Memory Surfaces

| Memory surface | What it preserves | Risk |
|---|---|---|
| Golden-age House of Wisdom | Abbasid scholarship as a civilizational symbol. | Turns distributed scholarship into one building. |
| River-of-ink catastrophe | Destroyed books made visible through image. | Converts literary memory into audited fact. |
| Lost universal library | Desire for one tragic knowledge-loss object. | Repeats Alexandria-style compression. |
| Modern institutional reuse of the name | Recovery, pride, education, and cultural aspiration. | Projects modern values backward into Abbasid categories. |

## RUINS Treatment

The House of Wisdom afterlife is part of the evidence history. RUINS should not
erase memory because it is not direct evidence. It should classify memory as
memory and refuse to let it settle institutional facts by itself.
