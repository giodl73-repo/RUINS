**Track:** Evidence and Absence

This package treats Hurrian Hymn H6 as a RUINS case about recovering performance
from notation. It separates clay tablet survival, song text, interval names,
number signs, tuning term, damaged readings, staff notation, tablaturization,
modern performances, and public "oldest song" memory.
