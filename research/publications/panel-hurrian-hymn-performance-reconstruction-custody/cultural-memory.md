# Cultural Memory Register

| Memory surface | RUINS handling |
|---|---|
| Oldest song | Public shorthand for notated survival, not proof of complete recovered music. |
| Playable melody | Modern interpretation layer, not direct ancient sound. |
| Viral recordings | Reception and pedagogy, not evidence of original performance. |
| Modern staff score | Accessibility layer that can over-close ambiguity. |
| Tablature | Better ambiguity-preserving representation, still not original performance. |
| Fertility or ritual framing | Possible text/ritual layer owned by RITE and text systems unless separately reviewed. |
