# Notated Performance Gate

| Layer | H6 custody question |
|---|---|
| Object | Does the claim stay anchored to the tablet and its condition? |
| Text | Does it separate hymn words from musical instructions? |
| Notation | Does it distinguish interval names, number signs, tuning terms, and damaged signs? |
| Reading | Does it separate sign reading from interpretation of function? |
| Translation | Does it keep translation separate from melody and performance claims? |
| Modern notation | Does it label staff notation as a modern presentation, not recovered sound? |
| Tablature | Does it preserve string-number ambiguity rather than pretending certainty? |
| Performance | Does it classify modern recordings as interpretations or afterlife? |
| Memory | Does "oldest song" language stay public-memory shorthand rather than proof of playability? |
