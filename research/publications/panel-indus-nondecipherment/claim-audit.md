# Claim Audit: Indus Script Non-Decipherment

**Owner:** ERRATA
**Status:** working audit

## Claim Grades

| Claim | Max grade now | Audit note |
|---|---|---|
| Many Indus signs survive on inscribed artifacts. | Observed | Direct material exists, though counts depend on cataloging. |
| The inscriptions are generally short. | Corroborated | Repeated in corpus/statistical discussions; exact values require source lock. |
| There is no generally accepted decipherment. | Corroborated | Treat as current scholarly status, not as proof of impossibility. |
| There is no known bilingual anchor comparable to a Rosetta Stone. | Corroborated | Strong reason for refusal discipline. |
| The system has statistical structure. | Plausible | Statistical studies support structure, not translation. |
| The system encodes a specific language family. | Speculative | Requires stronger linguistic bridge than current accepted evidence. |
| The system is definitely non-linguistic. | Speculative | Serious critique, but not settled by shortness alone. |
| A proposed decipherment solves the script. | Refuse | Out of scope unless treated as an object of review. |

## Refusals

- RUINS will not translate signs.
- RUINS will not choose a language family.
- RUINS will not treat statistical order as semantic reading.
- RUINS will not treat non-decipherment as total ignorance.
- RUINS will not turn "mystery" into evidence.

## Phrases To Challenge

| Phrase | Replacement |
|---|---|
| "the script says" | "the proposed reading claims" |
| "proves a language" | "supports structure consistent with..." |
| "meaningless symbols" | "possibly non-linguistic sign system" |
| "lost language solved" | "proposed decipherment not generally accepted" |
| "Rosetta Stone missing, so nothing can be known" | "no bilingual anchor, so translation claims remain weak" |
