# Source Map: Indus Script Non-Decipherment

**Owner:** PALIMPSEST
**Status:** working source map
**Rule:** metadata-only; do not copy protected source content

This map records source surfaces and source targets. It does not settle the
script's linguistic status.

## Evidence Surfaces

| Surface | Evidence condition | RUINS use |
|---|---|---|
| Seals, sealings, tablets, pottery marks, and other inscribed artifacts | direct material through cataloged artifacts and excavation reports | Establish what survives. |
| Corpus catalogs and sign lists | copied / reported catalog evidence | Count and classify signs without claiming meaning. |
| Archaeological context | contextual evidence | Connect inscriptions to places, objects, and possible uses. |
| Statistical analyses | reconstruction / analytic model | Test structure, directionality, and ordering without translating. |
| Decipherment proposals | speculative reconstruction | Classify claims and refusal boundaries. |
| Non-linguistic critiques | interpretive critique | Test whether writing assumptions are being overclaimed. |
| Public mystery narratives | cultural memory | Track how absence becomes popular meaning. |

## Source Targets

| Target | Use | Claim boundary |
|---|---|---|
| Parpola/Joshi/Pande corpus volumes and related catalog traditions | Corpus baseline and sign inventory history. | Cataloging signs is not decipherment. |
| Mahadevan concordance and sign-list tradition | Sign frequencies, variants, and concordance framing. | Frequencies support structure claims only. |
| Farmer, Sproat, and Witzel 2004 critique | Non-linguistic hypothesis and short-text argument. | Serious critique, not final proof of non-language. |
| Rao/Yadav/Joglekar/Vahia/Mahadevan/Adhikari statistical work | Order, directionality, and information-theoretic structure. | Structure is not translation. |
| Sinha/Ashraf/Pan/Wells network analysis | Syntactic-organization hypothesis. | Network pattern is not accepted reading. |
| Nair 2026 synthetic-baseline scorecard | Current contested-method update to non-linguistic baseline testing. | Preprint; useful for method pressure, not settled authority. |
| Recent public summaries of decipherment obstacles | Public-facing memory and current-state framing. | Treat as cultural-memory/current-summary evidence, not primary authority. |
