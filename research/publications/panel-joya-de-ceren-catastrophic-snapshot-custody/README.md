# Joya de Ceren Catastrophic Snapshot Custody

Status: publication-ready
Track: Evidence and Absence
Rubric: v1.18
Score: 46/50

Joya de Ceren is the domestic and agricultural countercase to Akrotiri's urban
ash-preserved settlement. It preserves a Classic-period farming village buried
by volcanic ash, including architecture, household spaces, agricultural
contexts, and activity surfaces that can be mistaken for complete Maya daily
life.
