# Catastrophic Village Snapshot Gate

## Gate Rule

Preserved village contexts may support unusually strong household and
agricultural claims, but they do not make the village a complete frozen record
of daily life.

## Evidence Layers

| Layer | Can support | Cannot support alone |
|---|---|---|
| Ash burial | Preservation and destruction context. | Complete final moment. |
| Structure context | Building function where source-locked. | Whole household history. |
| Object context | Activity surface or stored object category. | Exact last action. |
| Plant remains | Crop/field/food evidence where method-locked. | Complete diet or crop economy. |
| Field/garden evidence | Local agricultural practice. | Whole regional agriculture. |
| Absence of bodies | Evacuation question. | Certain escape story. |
| UNESCO status | Heritage value and official recognition. | Complete society or history. |
| Pompeii comparison | Public memory and interpretive pressure. | Evidence equivalence. |

## Publication Test

Before publication, each Joya de Ceren claim must answer:

1. Is this household, field, object, crop, destruction, absence, or memory
   evidence?
2. Is the evidence primary context, inferred use, preserved sample, or public
   interpretation?
3. Does the claim require CERES, food/agriculture, MAXIM/LUCIA, CANON, or
   medical/forensic handoff?
4. Is the exact claim being made, refused, or demoted to P3?
