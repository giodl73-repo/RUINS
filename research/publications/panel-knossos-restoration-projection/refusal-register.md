# Refusal Register: Knossos Restoration Projection

**Owner:** ERRATA / Safety Editor
**Status:** first register
**Date:** 2026-07-04

## Refused Claims

| Claim | Refusal | Allowed substitute |
|---|---|---|
| Restored Knossos is simply what Minoans built. | Visible surfaces include modern restoration, completion, and conservation. | Knossos is a Bronze Age site with extensive modern restoration layers. |
| Knossos is fake because Evans restored it. | Modern restoration does not erase ancient fabric or excavation value. | Ancient fabric and modern projection coexist. |
| Concrete columns and beams are Bronze Age originals. | Concrete is a modern restoration material. | Concrete is a visible restoration/reconstitution layer. |
| Restored frescoes are complete originals. | Fragment, completion, copy, and display must be separated. | Restored frescoes are source-specific fragment-plus-completion surfaces. |
| UNESCO validates Evans's restoration choices. | UNESCO recognition concerns serial heritage value and significance. | UNESCO supports heritage/custody framing only. |
| Minos, Labyrinth, or Minotaur explain the archaeology. | Myth is not direct site evidence. | Mythic language is cultural memory unless separately source-locked. |
| Digital models recover the palace exactly. | Models are modern reconstructions built from selected evidence and assumptions. | Digital models can help separate or display layers. |

## Phrase Controls

| Avoid | Use instead |
|---|---|
| "the Minoan palace as restored" | "Evans-era restoration layered onto Bronze Age remains" |
| "fake palace" | "ancient site with extensive modern reconstitution" |
| "original fresco" | "fresco fragment, restoration, replica, or completion, source-specific" |
| "UNESCO proves it" | "UNESCO recognizes the serial Minoan palatial property" |
| "the Labyrinth" | "Labyrinth-associated cultural memory" |
