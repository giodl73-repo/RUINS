# Context-Loss Gate: Kyrenia Shipwreck

**Owner:** ASHLAR / PALIMPSEST / ERRATA
**Status:** second-case test gate
**Date:** 2026-07-05

This gate tests whether `ruins-capability:shipwreck-context-loss` works for a
case where the central pressure is reconstruction and display rather than only
cargo abundance.

## Current Scope

| Claim surface | Current RUINS permission |
|---|---|
| Wreck identity and excavation frame | Allowed, source-scoped to INA/final-report metadata. |
| Hull preservation and reconstruction | Allowed as broad custody claim. |
| Cargo classes and dating evidence | Allowed as broad source-summary claims. |
| Exact object counts, IDs, cargo distributions, coins, stamps, or timbers | Refuse until final-report/catalogue page custody. |
| Route, home port, owner, crew, or final voyage closure | Refuse for current draft. |
| Replica sailing performance as ancient performance | Refuse without dedicated experimental custody. |
| Museum display or digital catalog as original stowage | Refuse. |
| Site-location, diving, recovery, collecting, or handling guidance | Refuse. |

## Core Test Question

If Uluburun proved the gate can handle spectacular cargo, Kyrenia asks whether
the same gate can handle reconstructed and catalogued survival. The answer is
yes only if RUINS keeps ancient evidence, excavation record, conservation,
reconstruction, replica, digital catalog, and new dating work separate.
