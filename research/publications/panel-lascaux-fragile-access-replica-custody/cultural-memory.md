# Cultural Memory: Lascaux

Lascaux's public life is dominated by masterpiece language, inaccessibility,
replica experience, virtual access, and the feeling of entering prehistory.

## Memory Surfaces

| Memory surface | RUINS handling |
|---|---|
| Prehistoric masterpiece | Reception and value language, not final interpretation. |
| Inaccessible original | Conservation/access condition, not total loss. |
| Exact replica | Visitor-access claim requiring replica custody. |
| Virtual cave | Digital access, not material presence. |
| Origins of art/humanity | Broad cultural memory unless separately sourced. |

## Rule

Public memory may explain why Lascaux matters now. It may not turn replicas or
virtual media into original-site evidence.
