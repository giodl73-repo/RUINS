# Chronology: Library of Alexandria Loss and Memory

**Owner:** ABSENCE HISTORIAN
**Status:** working chronology

This chronology separates events, reports about events, institutional decline,
and later memory. Dates are broad working anchors.

| Date / period | Layer | RUINS note |
|---|---|---|
| 3rd century BCE | Foundation / institution-building | Ptolemaic court patronage builds a research institution and book-collection memory. |
| 1st century BCE | Ancient institutional witness | Strabo describes the Museum as part of the royal palaces, with learned men and shared institutional life. |
| 48 BCE | Caesar's Alexandrian war | Fire traditions begin here, but sources differ on whether the Library, nearby warehouses, books, docks, or broader city areas burned. |
| 1st-3rd centuries CE | Decline / displacement problem | The Library's condition is not a single well-documented event. RUINS treats continuity, rebuilding, dispersal, and loss as separate questions. |
| 391/392 CE | Serapeum destruction context | The destruction of the Serapeum becomes a second major loss surface, but the relation between Serapeum holdings and the Ptolemaic Library requires caution. |
| 7th century CE | Arab conquest of Alexandria | Later medieval accounts blame Caliph Umar; RUINS treats this as a late source tradition unless directly audited. |
| Medieval-modern period | Cultural memory | The Library becomes a symbol for lost universal knowledge and civilizational catastrophe. |

## Process Model

The Library is not handled as one fire. RUINS tracks:

1. institutional creation,
2. ancient book collection and scholarly activity,
3. war damage traditions,
4. possible decline, dispersal, rebuilding, or institutional transformation,
5. Serapeum-era religious/political destruction,
6. later blame legends,
7. modern cultural-memory uses.
