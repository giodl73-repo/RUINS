# Claim Audit: Library of Alexandria Loss and Memory

**Owner:** ERRATA
**Status:** working audit

## Claim Grades

| Claim | Max grade now | Audit note |
|---|---|---|
| Alexandria had a major scholarly institution associated with the Museum. | Corroborated | Strabo is a strong ancient witness for the Museum setting. |
| The Library's exact holdings can be reconstructed. | Refuse | The surviving evidence does not support a holdings reconstruction. |
| Caesar definitely destroyed the entire Library in 48 BCE. | Speculative | Plutarch supports a destruction tradition, but other evidence complicates total-loss certainty. |
| Caesar's fire affected books or book-storage areas. | Plausible | Cassius Dio and later traditions support book-loss framing, but the exact institution and scale remain uncertain. |
| The Serapeum destruction was the destruction of the original Great Library. | Speculative | The relationship between Serapeum holdings and the Ptolemaic Library requires source-locked treatment. |
| Caliph Umar ordered the Library burned in 642 CE. | Refuse as historical claim in this draft | The story is late and should be treated as later source tradition until audited. |
| The Library is a modern symbol of lost knowledge. | Corroborated as cultural memory | Public and scholarly writing repeatedly use it as an emblem of knowledge loss. |

## Phrases To Challenge

| Phrase | Replacement |
|---|---|
| "the Library burned down" | "one destruction tradition attributes damage to..." |
| "the ancient world's knowledge was lost in one fire" | "the Library became a symbol for lost knowledge" |
| "Caesar destroyed the Library" | "Plutarch reports a Caesar-fire destruction tradition" |
| "the Muslims burned the Library" | "later medieval accounts attribute destruction to Umar; this draft refuses that as direct evidence" |
| "the Library contained all ancient knowledge" | "the Library is remembered as a universal-library ideal" |
