# Source Lock: Library of Alexandria Loss and Memory

**Owner:** PALIMPSEST
**Status:** first source lock
**Date:** 2026-07-04

This file locks the first source set for the Alexandria working paper. Locking a
source means RUINS may use it for bounded claims. It does not mean the source
settles the Library's final fate.

## Locked Sources

|---|---|---|---|---|

## Locked Values And Claims

| Claim | Locked wording | Source | Grade |
|---|---|---|---|
| Museum institutional setting | Strabo describes the Museum as part of the royal palaces, with institutional spaces and a community of learned men. | S1 | Corroborated ancient witness |
| Museum is not Library fate | Strabo's Museum description cannot by itself settle the fate of the Library collection. | S1 | Refusal support |
| Plutarch's Caesar-fire tradition | Plutarch reports a tradition in which fire used during Caesar's Alexandrian war spread from the dockyards and destroyed the great library. | S2 | Corroborated as narrative tradition |
| Cassius Dio's fire account | Cassius Dio reports many fires and says docks, grain storehouses, other buildings, and a library/books were burned. | S3 | Corroborated as alternative narrative tradition |
| One-fire certainty | The first source set does not support reducing the Library's fate to one certain destruction event. | S1 / S2 / S3 | Refusal support |
| Modern synthesis target | El-Abbadi's *Life and Fate* is a major modern source target with UNESCO/UN bibliographic custody. | S4 | Bibliographic custody |
| Serapeum debate | Serapeum/library claims require a dedicated source lock before this paper can treat them as more than a working question. | S5 | P2 source need |
| Public Library myth | Popular writing often treats Alexandria as a symbol of lost universal knowledge. | S6 | Cultural-memory evidence |

## Paper Use Rules

1. Say "Plutarch reports" or "Cassius Dio reports" when describing Caesar-fire
   traditions.
2. Do not merge Plutarch and Cassius Dio into one harmonized event account.
3. Do not infer full Library destruction from a report of books, storehouses, or
   a library burning.
4. Do not equate Museum, Library, Serapeum, and book warehouses without a
   separate identity audit.
5. Treat the Arab-conquest story as a late source tradition until direct source
   custody is added.
6. Treat public summaries as cultural-memory evidence only.
