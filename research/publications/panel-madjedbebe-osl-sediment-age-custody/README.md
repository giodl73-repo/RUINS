# Madjedbebe OSL Sediment-Age Custody

**Track:** Evidence and Absence
**Status:** publication-ready
**Rubric:** v1.51
**Score:** 46/50
**Capability decision:** promote `ruins-capability:osl-sediment-age-custody` to Core

This package uses Madjedbebe optically stimulated luminescence dating as the
trigger case for OSL sediment-age custody. It keeps sediment samples, grain
populations, bleaching, burial context, dose models, dose rates, stratigraphy,
artefact association, occupation inference, regional synthesis, and public
first-arrival memory separate.
