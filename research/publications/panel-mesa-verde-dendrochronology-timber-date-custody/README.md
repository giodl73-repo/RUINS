# Mesa Verde Dendrochronology Timber-Date Custody

**Track:** Evidence and Absence
**Status:** publication-ready
**Rubric:** v1.53
**Score:** 46/50
**Capability decision:** promote `ruins-capability:dendrochronological-timber-date-custody` to Core

This package uses Mesa Verde and Southwest tree-ring archaeology as the trigger
case for dendrochronological timber-date custody. It keeps samples, trees, ring
sequences, terminal rings, felling, reuse, construction context, repair episodes,
reference chronologies, provenance, drought signals, occupation inference, and
public exact-year memory separate.
