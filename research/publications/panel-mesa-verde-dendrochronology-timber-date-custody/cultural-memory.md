# Cultural Memory Register

Tree-ring precision creates public-memory pressure because annual dates sound
final. This paper contains that pressure.

| Memory surface | Handling |
|---|---|
| Exact-year building story | Allowed only as dated timber unless construction context is locked. |
| Cliff-dwelling chronology | Treated as public interpretation, not total settlement history. |
| Drought-collapse narrative | Climate signal is handed to climate systems and social causation to MAXIM/LUCIA. |
| Vanished-community story | Kept as afterlife memory unless occupation and migration evidence is separately locked. |
| Timber-trade story | Provenance evidence cannot become complete supply network without additional custody. |
