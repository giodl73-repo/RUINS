# Microbotanical Starch-Phytolith Custody

**Track:** Evidence and Absence
**Status:** publication-ready
**Rubric:** v1.56
**Score:** 46/50
**Capability decision:** promote `ruins-capability:microbotanical-starch-phytolith-custody` to Core

This package uses archaeological starch, phytolith, and plant-residue studies as
the trigger case for microbotanical custody. It keeps samples, microremain type,
preservation, contamination, extraction, reference collections, taxonomic
assignment, processing damage, context, residue formation, quantification,
activity inference, diet/domestication synthesis, and public plant-use memory
separate.
