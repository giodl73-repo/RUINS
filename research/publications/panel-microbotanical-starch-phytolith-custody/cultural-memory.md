# Cultural Memory Register

Plant microremains create public-memory pressure because small traces can be
turned into vivid ancient meals, first crops, recipes, or medicines. This paper
contains that pressure.

| Memory surface | Handling |
|---|---|
| Ancient meal | Refused from isolated microremains. |
| First crop | Requires independent dating, domestication, and regional custody. |
| Complete diet | Handed to MAXIM/LUCIA after multiple evidence lines. |
| Tool function | Requires use-wear, context, and residue formation custody. |
| Recipe or medicine | Refused without textual, experimental, and ingredient custody. |
| Seasonal calendar | Requires sampling design and independent seasonality evidence. |
