# Refusal Register: Mound Builder Denial

**Owner:** ERRATA / Safety Editor
**Status:** publication-ready register
**Date:** 2026-07-05

## Refused Claims

| Claim | Refusal | Allowed substitute |
|---|---|---|
| A vanished non-Native race built the mounds. | This is Mound Builder denial mythology, not current archaeology. | Use source-scoped Indigenous / American Indian authorship. |
| Giants built the mounds. | Giant-builder claims are pseudoarchaeological and deny Indigenous achievement. | Discuss giant stories only as public myth or denial afterlife. |
| Ancient astronauts built the mounds. | This is modern pseudoarchaeology, not evidence. | Earthworks show Indigenous engineering, ceremony, and landscape knowledge. |
| Israelites, Vikings, Welsh, Romans, Toltecs, or other visitors built the mounds. | Visitor-builder theories are speculative denial unless separately source-locked as historiography. | Treat them as denial narratives or antiquarian speculation. |
| Impressive geometry proves outsiders. | Geometry is not authorship proof. | Attribute geometry as source-scoped Indigenous monumental practice. |
| Imported or distant materials prove foreign builders. | Long-distance materials show exchange, gathering, or contact, not authorship. | Use contact/exchange language. |
| "Mysterious mounds" as default framing. | Mystery language can revive denial by implying local people could not have built them. | Use "monumental Indigenous earthworks with bounded unanswered questions." |
| Modern sacred memory proves every archaeological detail. | Living memory and archaeology answer different claim types. | Treat sacred memory as custody/afterlife and keep site claims source-scoped. |
| Site details can be used for collecting or excavation. | RUINS does not provide looting, collecting, excavation, or disturbance guidance. | Keep all guidance interpretive and non-operational. |

## Phrase Controls

| Avoid | Use instead |
|---|---|
| "mysterious Mound Builders" | "historic Mound Builder denial narrative" |
| "lost race" | "vanished-race story, refused as archaeology" |
| "no one knows who built them" | "current sources attribute this site or complex to Indigenous/American Indian builders, with bounded site-specific debates" |
| "advanced outsiders" | "Indigenous monumental architecture and landscape knowledge" |
| "ancient secrets" | "public mystery framing or pseudoarchaeological afterlife" |
