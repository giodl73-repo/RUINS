# Cultural Memory Register

Obsidian sourcing creates vivid public maps because source-to-site lines look
like ancient routes. This paper contains that pressure.

| Memory surface | Handling |
|---|---|
| Ancient trade map | Allowed only as public visualization unless exchange mechanism is locked. |
| Migration path | Refused from source distance alone. |
| Origin story | Source is glass origin, not people origin. |
| Ethnic or territorial map | Handed to identity systems and refused without independent evidence. |
| Market network | Requires economic synthesis outside RUINS. |
| Political alliance | Requires historical and social evidence beyond obsidian clusters. |
