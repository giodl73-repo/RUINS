# Paradata Transparency Gate

| Layer | Oplontis custody question |
|---|---|
| Publication | Which digital volume, chapter, figure, or database record anchors the claim? |
| Model | Which model version, room, state, and navigation mode is being used? |
| Fragment | Which painting, stucco, pavement, sculpture, or architectural fragment is placed? |
| State | Is the view actual, restored, hypothetical, comparative, or didactic? |
| Decision | Are placement, height, roof, wall, light, and decor choices documented? |
| Uncertainty | Are missing, inferred, and alternative reconstructions visible? |
| Evaluation | Can another scholar trace the claim back to source and decision records? |
| Communication | Is public navigation separated from evidential certainty? |
