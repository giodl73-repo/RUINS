# Plan

## Question

Can paradata transparency custody improve scoring for a concrete virtual
reconstruction project rather than only a guideline document?

## Scope

- Use Oplontis Project, King's Visualisation Lab, Cambridge review, and project
  publication materials.
- Separate database, model, publication, fragments, restored states, lighting,
  viewing, public access, and paradata sufficiency.
- Promote the capability to Core with rubric v1.34 and a checklist.

## Out Of Scope

- No evaluation of every Oplontis room, object, or fragment.
- No implementation review of Unity or database internals.
- No claim that digital reconstruction proves original experience.
