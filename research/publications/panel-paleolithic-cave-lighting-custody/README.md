# Paleolithic Cave Lighting Custody

**Track:** Evidence and Absence
**Status:** publication-ready
**Rubric:** v1.31
**Score:** 45/50
**Capability decision:** activate `ruins-capability:light-mediated-reconstruction-custody`

This package treats experimental archaeology of Paleolithic cave lighting as a
RUINS case for reconstructed visual conditions. It keeps archaeological traces,
experimental fuels, flame behavior, smoke, movement, visibility, and image-use
claims in separate custody.
