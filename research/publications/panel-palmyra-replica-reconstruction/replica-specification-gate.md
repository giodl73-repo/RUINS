# Replica Specification Gate: Palmyra Replica Reconstruction

**Owner:** PALIMPSEST / ERRATA
**Status:** current-scope replica specification gate
**Date:** 2026-07-05

This gate separates project-public replica descriptions from engineering,
conservation, structural, and original-monument claims.

## Current Permissions

| Specification surface | Current permission |
|---|---|
| Saying the public arch was a scale facsimile/replica | Allowed, source-scoped to IDA/Oxford. |
| Saying Oxford describes it as a 1:3 scale facsimile | Allowed, source-scoped to Oxford. |
| Saying IDA media describes a 13-tonne replica | Allowed, source-scoped to IDA media. |
| Saying project sources describe photogrammetry-based modelling and machining | Allowed, source-scoped to IDA/Oxford. |
| Saying project sources describe stone or marble machining | Allowed as project-public description, not conservation material analysis. |
| Giving precise engineering dimensions, tolerances, quarry/source, toolpaths, or structural claims | Refuse until direct technical/project documentation is added. |
| Treating replica specs as original-monument specs | Refuse. |
| Treating replica specs as on-site restoration methodology | Refuse. |

## Current Decision

The project-public replica specification P2 is closed for current RUINS wording.
Future exact technical, conservation, structural, or engineering claims require
separate project-document custody.
