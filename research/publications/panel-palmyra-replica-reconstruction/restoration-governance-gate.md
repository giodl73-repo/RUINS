# Restoration Governance Gate: Palmyra Replica Reconstruction

**Owner:** PALIMPSEST / ASHLAR
**Status:** current governance gate closed
**Date:** 2026-07-05

This gate blocks reconstruction claims that outrun UNESCO and World Heritage
Committee custody.

## Current Permissions

| Claim type | Current permission |
|---|---|
| Naming UNESCO OUV and Danger-list status | Allowed. |
| Naming Arch of Triumph destruction | Allowed. |
| Naming the IDA/Oxford replica as a public replica/facsimile | Allowed. |
| Saying the replica restored the original arch | Refuse. |
| Saying Palmyra has been rebuilt | Refuse. |
| Saying a digital model authorizes site reconstruction | Refuse. |
| Making conservation or engineering recommendations | Refuse. |
