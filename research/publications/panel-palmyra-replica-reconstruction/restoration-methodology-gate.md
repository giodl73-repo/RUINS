# Restoration Methodology Gate: Palmyra Replica Reconstruction

**Owner:** PALIMPSEST / ERRATA / ASHLAR
**Status:** current-scope gate closed
**Date:** 2026-07-05

This gate closes the remaining P2 for the current paper scope by refusing
on-site restoration methodology claims. The paper is publication-ready as a
RUINS replica-status methodology case because it does not publish conservation
methods, engineering plans, material recommendations, or authorized
reconstruction procedures.

## Current Scope

| Claim surface | Current RUINS permission |
|---|---|
| UNESCO recovery governance | Allowed when source-scoped to UNESCO and World Heritage Committee records. |
| Emergency-first / assessment-before-reconstruction posture | Allowed when source-scoped to UNESCO recovery guidance. |
| IDA/Oxford public replica specifications | Allowed only as project-public replica description. |
| On-site restoration materials, dimensions, techniques, or sequencing | Refuse. |
| Engineering or conservation recommendations | Refuse. |
| Claim that the replica authorizes reconstruction | Refuse. |
| Claim that Palmyra has been rebuilt | Refuse. |

## Future Methodology Minimum

Before RUINS adds any on-site restoration-methodology claim, it must record:

1. Feature or intervention scope.
2. State Party / UNESCO / World Heritage Committee custody.
3. Field assessment source.
4. Restoration or reconstruction methodology source.
5. Materials and structural evidence source, if any.
6. Whether the claim is documentation, emergency stabilization, restoration,
   reconstruction, replica production, or public memory.

## Decision

No open P2 remains for the current paper. Restoration methodology remains a P3
future-use gate, not a blocker for this non-intervention methodology paper.
