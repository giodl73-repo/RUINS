# Parchment Biocodicology Custody

**Track:** Evidence and Absence
**Status:** publication-ready
**Rubric:** v1.42
**Score:** 46/50
**Capability decision:** promote `ruins-capability:parchment-biocodicology-custody` to Core

This package uses parchment biocodicology and ZooMS as the trigger case for
manuscript biomolecular custody. It keeps eraser-crumb sampling, collagen
peptide mass fingerprints, DNA, microbiome, species calls, folio scale,
production history, library reconstruction, textual authority, and public
living-book memory separate.
