# Dispersed Object Custody Gate: Parthenon Sculptures

**Owner:** PALIMPSEST / ERRATA
**Status:** current-scope gate closed
**Date:** 2026-07-05

This gate prevents a surviving-sculpture paper from silently becoming a legal
decision, diplomatic settlement, complete reconstruction, or replica-equivalence
claim.

## Current Permissions

| Claim type | Current permission |
|---|---|
| British Museum holding summary | Allowed, source-scoped. |
| Acropolis Museum gallery strategy | Allowed, source-scoped. |
| Elgin removal narrative | Allowed as British Museum-source provenance framing, not adjudication. |
| Greek return request | Allowed as request layer. |
| Loan/title/public position | Allowed as institutional position layer. |
| Plaster copies | Allowed as display/absence markers. |
| Legal title decision | Refuse. |
| Complete monument reconstruction | Refuse. |
| Plaster copy or digital record as object-equivalent | Refuse. |
| Final diplomatic settlement | Refuse. |

## Future Exact-Claim Minimum

Before RUINS makes stronger claims, it must record object identifiers,
catalogue records, removal/acquisition documents, legal/policy custody,
conservation status, and sibling handoffs for history, law, geography, object
identity, and material interpretation.
