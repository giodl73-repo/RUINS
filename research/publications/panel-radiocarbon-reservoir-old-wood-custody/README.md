# Radiocarbon Reservoir Old-Wood Custody

**Track:** Evidence and Absence
**Status:** publication-ready
**Rubric:** v1.55
**Score:** 46/50
**Capability decision:** promote `ruins-capability:radiocarbon-reservoir-old-wood-custody` to Core

This package uses radiocarbon reservoir effects, old-wood risk, and Bayesian
chronological modelling as the trigger case for radiocarbon-reservoir custody.
It keeps sample material, organism life, carbon source, context, pretreatment,
measurement, calibration, reservoir model, old-wood risk, dietary offset,
Bayesian priors, model output, event inference, and public date memory separate.
