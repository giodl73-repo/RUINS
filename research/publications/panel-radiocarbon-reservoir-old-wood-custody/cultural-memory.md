# Cultural Memory Register

Radiocarbon dates become public memory when a calibrated range is simplified to
a single calendar year or when model output is used as a solved chronology.
This paper contains that pressure.

| Memory surface | Handling |
|---|---|
| Exact calendar date | Refused unless event association is separately locked. |
| Oldest-first headline | Treated as public framing, not chronology proof. |
| Solved chronology | Refused from model output alone. |
| Collapse timing | Handed to MAXIM/LUCIA after event and process custody. |
| Diet proof | Requires organism, collagen, isotope, and reservoir custody. |
| Population curve | Requires demographic model outside RUINS. |
