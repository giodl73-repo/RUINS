# Cultural Memory Register

| Memory surface | RUINS handling |
|---|---|
| Ecocide | Public moral memory, not direct proxy conclusion. |
| Last tree | Narrative shorthand requiring taxon, chronology, and scale custody. |
| Collapse | Social-history claim requiring demography and archaeology. |
| Resilience | Counter-memory also requiring source custody. |
| Human overuse | Attribution claim, not pollen decline by itself. |
| Climate stress | Environmental pressure, not cultural determinism. |
