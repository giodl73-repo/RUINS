# Lake-Core Proxy Gate

| Layer | Lake-core proxy custody question |
|---|---|
| Core | Which lake, peat, catchment, depth, sample interval, and archive condition produced the signal? |
| Catchment | Is the proxy local basin, crater, wetland, regional vegetation, or island-wide inference? |
| Proxy | Is the signal pollen, spores, charcoal, diatoms, phytoliths, sediment, geochemistry, or isotope value? |
| Preservation | What mixing, hiatus, erosion, inwash, preservation, and sampling controls are locked? |
| Chronology | Which radiocarbon dates, age model, reservoir effect, interpolation, and uncertainty support timing? |
| Taxon | Is the identification family, genus, species, vegetation class, crop, weed, palm, grass, or unknown? |
| Fire | Is charcoal local burning, regional fire, natural ignition, human clearance, or redeposited material? |
| Attribution | Are human activity, rats, drought, climate, erosion, and vegetation dynamics separately tested? |
| Scale | Is the claim core event, catchment trend, island pattern, land use, demography, or social history? |
| Memory | Are ecocide, last-tree, collapse, resilience, and moral-parable stories contained? |
