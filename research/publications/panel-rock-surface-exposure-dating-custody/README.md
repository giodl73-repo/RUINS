# Rock-Surface Exposure Dating Custody

**Track:** Evidence and Absence
**Status:** publication-ready
**Rubric:** v1.64
**Score:** 46/50
**Capability decision:** promote `ruins-capability:rock-surface-exposure-dating-custody` to Core

This package uses cosmogenic nuclide dating, lichenometry, rock varnish, and
microerosion papers as the trigger case for surface-exposure custody. It keeps
sample surfaces, exposure events, inheritance, shielding, erosion, calibration,
species/substrate, varnish formation, disturbance, date models, archaeological
association, event inference, and public age memory separate.
