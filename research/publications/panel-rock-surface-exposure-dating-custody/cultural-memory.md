# Cultural Memory

Surface dating creates public-memory pressure because a single age estimate can
look like the date of a monument or carving.

| Memory claim | Handling |
|---|---|
| Oldest monument | Require construction and association custody beyond surface exposure. |
| Direct carving date | Require carving-surface and post-carving formation controls. |
| Solved petroglyph age | Treat as public headline unless calibration and association are locked. |
| Cultural phase date | Hand to MAXIM/LUCIA unless independent phase evidence is locked. |
| Abandonment event | Require site-process evidence beyond exposure age. |
| Universal method | Refuse; calibration and local conditions remain visible. |
