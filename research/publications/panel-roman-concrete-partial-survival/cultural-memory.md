# Cultural Memory Register: Roman Concrete Partial Survival

**Owner:** Cultural Memory Reader
**Status:** first register
**Date:** 2026-07-04

## Memory Forms

| Memory form | Evidence surface | RUINS treatment |
|---|---|---|
| "Secret of Roman concrete" | Popular science and institutional summary language. | Treat as public compression of multiple source-specific mechanisms. |
| "Lost recipe rediscovered" | Media narrative around modern materials papers. | Replace with partial survival: text, material, sample, and model. |
| "Roman concrete is better than modern concrete" | Civilizational nostalgia and simplified durability comparisons. | Refuse category-wide comparison across environments and requirements. |
| "Self-healing miracle" | Public retelling of lime-clast and hot-mixing mechanisms. | Use only with sample/model labels. |
| Green concrete future | Modern sustainability and engineering translation. | Treat as modern afterlife, not ancient intent. |
| Vitruvius as total authority | Renaissance and modern reverence for the ancient technical text. | Treat as important witness, not exhaustive manual. |

## Publication Rules

1. Put "secret," "miracle," and "lost recipe" language in critique or
   cultural-memory context.
2. Do not use public summaries as authority for material mechanisms when a
   primary paper is available.
3. Treat modern sustainability claims as translation, not recovery.
4. Separate ancient practice from modern Roman-inspired concrete.
