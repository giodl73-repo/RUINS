**Track:** Evidence and Absence

**Score:** 46/50
**Capability decision:** promote `ruins-capability:cryptotephra-marker-horizon-custody` to Core

This package uses Santorini/Thera eruption chronology and cryptotephra debates
as the trigger case for marker-horizon custody. It keeps glass shards, ash
chemistry, sulfate spikes, archive correlations, stratigraphy, radiocarbon and
archaeological chronologies, eruption attribution, cultural synchronization, and
public eruption/collapse memory separate.
