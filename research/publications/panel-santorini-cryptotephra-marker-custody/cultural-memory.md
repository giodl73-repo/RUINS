# Cultural Memory Register

| Memory surface | RUINS handling |
|---|---|
| Exact eruption year | Public chronology memory, not direct shard evidence. |
| Santorini solves Bronze Age dating | Synchronization memory requiring sibling custody. |
| Ash layer equals destruction | Catastrophe shorthand requiring site-level evidence. |
| Greenland proves Thera | Source-attribution memory requiring geochemical match. |
| Atlantis or collapse stories | Public myth/collapse memory, not archaeological conclusion. |
| Climate forcing event | Climate-system claim requiring separate model custody. |
