# Shell Midden Sclerochronology Seasonality Custody

**Track:** Evidence and Absence
**Status:** publication-ready
**Rubric:** v1.52
**Score:** 46/50
**Capability decision:** promote `ruins-capability:shell-sclerochronology-seasonality-custody` to Core

This package uses shell midden sclerochronology and oxygen-isotope seasonality
as the trigger case for shell-seasonality custody. It keeps shells, growth
increments, isotope systems, shell edges, habitat baselines, cooking alteration,
death-season models, assemblage sampling, occupation inference, and public
seasonal-lifeway memory separate.
