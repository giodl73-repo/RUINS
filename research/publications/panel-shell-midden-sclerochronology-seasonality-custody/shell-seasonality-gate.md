# Shell Seasonality Gate

| Layer | Shell-seasonality custody question |
|---|---|
| Specimen | Which shell, taxon, element, layer, context, and collection history is measured? |
| Growth | Are increments continuous, seasonal, disturbed, repaired, old, juvenile, or edge-preserved? |
| Isotope | Is the signal oxygen, carbon, clumped isotope, trace element, mixed proxy, or altered carbonate? |
| Habitat | What water temperature, salinity, tidal zone, estuary/open-coast setting, and ecology are locked? |
| Baseline | Which modern or ancient water baseline, species calibration, and uncertainty supports seasonality? |
| Alteration | Are diagenesis, cooking, burning, boiling, roasting, and post-depositional change separated? |
| Death season | Is the claim shell death season, harvest season, deposition season, or site occupation season? |
| Assemblage | How many shells, contexts, taxa, years, and sampling biases support the pattern? |
| Synthesis | Are diet, mobility, settlement, climate, economy, and resource pressure handed to siblings? |
| Memory | Are seasonal camp, year-round occupation, ancient menu, and climate-record stories contained? |
