# Soil Micromorphology Occupation-Surface Custody

**Track:** Evidence and Absence
**Status:** publication-ready
**Rubric:** v1.58
**Score:** 46/50
**Capability decision:** promote `ruins-capability:soil-micromorphology-occupation-surface-custody` to Core

This package uses soil micromorphology, occupation surfaces, and experimental
floor-formation studies as the trigger case for micromorphology-surface custody.
It keeps block samples, contexts, microfacies, sediment sources, deposition,
post-depositional alteration, activity residues, floor renewal, cleaning/removal,
feature boundaries, spatial sampling, analogues, behavior inference, and public
domestic memory separate.
