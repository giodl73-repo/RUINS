# Cultural Memory Register

Micromorphology creates public-memory pressure because microscopic residues can
be turned into vivid domestic scenes. This paper contains that pressure.

| Memory surface | Handling |
|---|---|
| Domestic scene | Refused from a thin section alone. |
| Complete room function | Requires spatial and multiproxy custody. |
| Tidy house or cleaning routine | Absence is not cleaning without removal and preservation controls. |
| Workshop | Requires tool, residue, spatial, and production custody. |
| Animal pen | Requires dung source, distribution, and architectural custody. |
| Ritual deposit | Handed to RITE and refused without ritual evidence. |
