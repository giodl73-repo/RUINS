# Stone Tool Use-Wear Residue Custody

**Track:** Evidence and Absence
**Status:** publication-ready
**Rubric:** v1.43
**Score:** 46/50
**Capability decision:** promote `ruins-capability:lithic-usewear-residue-custody` to Core

This package uses stone-tool use-wear and residue analysis as the trigger case
for lithic surface-trace custody. It keeps edge traces, residue morphology,
surface texture, raw material, cleaning, taphonomy, experimental analogues,
worked-material inference, tool function, task scale, and public toolmaker
stories separate.
