# Cultural Memory Register

| Memory surface | RUINS handling |
|---|---|
| Tool tells us what happened | Public shorthand, not direct event record. |
| Hunter's knife | Hunting memory requiring independent custody. |
| Ancient diet from residue | Diet memory, not full diet. |
| Expert craftsperson | Skill biography, not surface trace alone. |
| Complete activity area | Site-activity synthesis outside trace-only evidence. |
| Experimental replica proves the past | Analogue memory, not direct ancient sequence. |
