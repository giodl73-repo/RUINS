# Lithic Use-Wear Gate

| Layer | Use-wear custody question |
|---|---|
| Artefact | Which tool, assemblage, context, date, raw material, and edge are claimed? |
| Edge | Is the trace polish, striation, rounding, scarring, residue, texture, or mixed evidence? |
| Raw material | How do grain, fracture, cortex, weathering, and heterogeneity affect the trace? |
| Taphonomy | Did trampling, sediment, handling, curation, or conservation modify the surface? |
| Cleaning | Were washing, brushing, solvents, storage, and contamination risks recorded? |
| Formation | Is the trace use, hafting, manufacture, retouch, prehension, or residue transfer? |
| Assignment | Are starch, phytolith, blood, collagen, plant, animal, and mineral residues separated? |
| Experiment | Which replica, motion, material, duration, force, and reference collection is used? |
| Action | Is the inference cutting, scraping, drilling, hide work, plant work, bone work, or mixed? |
| Scale | Is the claim edge, tool, assemblage, site activity, or social behavior scale? |
