# Urban Overbuild Window Gate

| Layer | Can support | Cannot support alone |
|---|---|---|
| Buried city frame | Overbuilt survival condition. | Complete Tenochtitlan reconstruction. |
| Excavation window | Presence and condition of exposed remains. | Whole building plan or full precinct inventory. |
| Under-building rescue | Discovery context and modern-construction mediation. | Claim that all related structures are known. |
| Removed museum object | Object survival and current custody. | Original context unless excavation custody is locked. |
| Colonial layer | Reuse, disturbance, and later occupation. | Simple destruction story or clean stratigraphy. |
| Conservation monitoring | Current preservation attention. | Finished conservation or future stability. |
| Digital scan / replica | Documentation and access layer. | Object equivalence or complete context. |
| Public rediscovery memory | Modern afterlife and national attention. | Original ritual meaning or historical completeness. |
