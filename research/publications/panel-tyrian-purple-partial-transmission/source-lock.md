# Source Lock: Tyrian Purple Partial Transmission

**Owner:** PALIMPSEST
**Status:** first source lock
**Date:** 2026-07-05

This source lock permits broad, source-scoped claims about murex-purple
technical survival. It does not authorize exact recipes, operational
production guidance, universal process claims, production scale, status rules,
or modern reconstruction equivalence.

## Paper Use Rules

1. Treat shell heaps as production evidence only when joined to context and
   analysis.
2. Treat chemical identification as dye/source evidence, not full recipe.
3. Treat ancient or museum process summaries as process fragments, not
   operational instructions.
4. Treat royal-purple memory as afterlife/status pressure, not universal use.
5. Do not provide modern harvesting, extraction, dyeing, or manufacturing
   guidance.
