# Technique Custody Gate: Tyrian Purple

**Owner:** ASHLAR / PALIMPSEST / ERRATA
**Status:** second-case test gate
**Date:** 2026-07-05

This gate tests `ruins-capability:partial-technical-transmission` outside Roman
concrete. It separates evidence that a technique existed from claims that the
complete ancient process has been recovered.

## Current Scope

| Claim surface | Current RUINS permission |
|---|---|
| Murex-purple production or use in specific sourced contexts | Allowed, source-scoped. |
| Shell deposits, pigment residues, HPLC markers, and textiles | Allowed as separate evidence layers. |
| Public summary of costly snail-derived process | Allowed as public/memory claim. |
| Exact ancient recipe or universal production sequence | Refuse. |
| Exact quantities, vessel parameters, temperatures, timing, and labor model | Refuse. |
| Modern harvesting, extraction, dyeing, or manufacturing guidance | Refuse. |
| Social status from color alone | Refuse. |
| One site/sample/textile as whole ancient industry | Refuse. |

## Future Exact-Claim Minimum

Before RUINS makes stronger technical claims, it must record:

1. Whether the claim comes from text, shell, pigment, textile, experiment, or
   public memory.
2. Site/sample/object custody.
3. Analytical method or textual passage.
4. Whether the claim is ancient observation, modern reconstruction, chemical
   identification, or public synthesis.
5. Sibling handoff if the claim belongs to CERES, FONTES, CANON, PORTO, MAXIM,
   or LUCIA.
