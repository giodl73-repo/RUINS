# Zooarchaeological Bone-Modification Custody

**Track:** Evidence and Absence
**Status:** publication-ready
**Rubric:** v1.57
**Score:** 46/50
**Capability decision:** promote `ruins-capability:zooarchaeological-bone-modification-custody` to Core

This package uses cut marks, tooth marks, trampling, percussion damage, burning,
and mortality profiles as the trigger case for bone-modification custody. It
keeps specimens, taxa, skeletal elements, surface preservation, mark types,
actors, formation processes, timing, assemblage sampling, age profiles,
quantification, experimental comparison, behavior inference, and public
subsistence memory separate.
