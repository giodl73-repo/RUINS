# Cultural Memory Register

Bone-modification evidence creates public-memory pressure because marks on bones
are easy to convert into vivid scenes of hunting, feasting, scavenging, or human
origins. This paper contains that pressure.

| Memory surface | Handling |
|---|---|
| Meat-eating origin story | Refused from marks alone. |
| Hunting scene | Requires assemblage, weapon, context, and behavior custody. |
| Scavenging certainty | Requires competing actor and access-sequence controls. |
| Ritual feast | Handed to RITE and refused without ritual evidence. |
| Complete diet | Requires multiple lines beyond bone counts. |
| Kill site | Requires spatial, depositional, and assemblage custody. |
