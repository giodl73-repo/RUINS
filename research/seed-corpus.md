# RUINS Seed Corpus

This is a candidate case list for the first RUINS research pass. It is not a
claim that each case is understood, equivalent, or ready for publication.

## Selection Principle

The seed set should cover different kinds of partial survival:

- destroyed or dispersed archives,
- abandoned cities and settlements,
- fragmentary texts,
- lost techniques,
- damaged machines or instruments,
- erased institutions,
- contested reconstructions,
- sites with strong later myth.

## Candidate Cases

| Case | Type | Initial RUINS question |
|---|---|---|
| Library of Alexandria | Destroyed/dispersed archive | How should RUINS handle a famous loss whose event history is more complicated than the myth? |
| House of Wisdom, Baghdad | Destroyed institution / contested memory | What survives as institutional memory versus direct source evidence? |
| Cahokia | Abandoned urban system | What can be inferred from earthworks, settlement pattern, and material remains without overnarrating collapse? |
| Great Zimbabwe | Monumental site / contested interpretation | How did later political and colonial narratives distort interpretation of surviving structures? |
| Indus script | Fragmentary text system | What does responsible non-decipherment look like as a knowledge record? |
| Antikythera mechanism | Damaged technical artifact | How should RUINS separate surviving mechanism, modern imaging, and reconstructed operation? |
| Roman concrete recipes | Partially surviving technique | When is a lost technique actually lost, partially documented, or experimentally rediscovered? |
| Vinland / Norse North Atlantic traces | Site evidence / saga memory | How should physical traces and literary memory be compared without collapsing them? |
| Knossos and Minoan reconstruction | Site restoration / modern projection | Where does restoration become a modern artifact with its own evidence status? |
| Dead Sea Scrolls fragments | Fragmentary textual corpus | How should fragment custody, reconstruction, translation, and authenticity concerns be represented? |
| Maya codices | Destroyed textual tradition | How should a corpus be described when only a small number of witnesses survive after deliberate destruction? |
| Palmyra | Damaged heritage site / digital reconstruction | How should RUINS grade post-destruction reconstruction, replicas, and cultural-memory claims? |

## First Five For Paper Work

1. Indus script - cleanest test of non-decipherment discipline.
2. Antikythera mechanism - strong test of artifact/reconstruction boundaries.
3. Library of Alexandria - strong test of myth versus historical loss.
4. Cahokia - strong test of abandoned urban systems without simple collapse
   stories.
5. Roman concrete recipes - strong test of lost technique versus partial
   survival.
