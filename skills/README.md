# RUINS Skills

Reusable paper and review stages for RUINS. Each skill lives in its own
directory as `SKILL.md`. Skills operationalize the review roles, quality rubric,
source posture, and forward-only method evolution.

## Paper Skills

| Skill | Purpose |
|---|---|
| [ruins-paper](ruins-paper/SKILL.md) | Scaffold or extend a RUINS paper package with the required research surfaces. |
| [ruins-source-lock](ruins-source-lock/SKILL.md) | Build a source lock and page-anchor audit for a paper's current claims. |

## Review Skills

| Skill | Purpose |
|---|---|
| [ruins-role-review](ruins-role-review/SKILL.md) | Run the RUINS role panel and score a paper against `QUALITY_RUBRIC.md`. |
| [ruins-method-evolve](ruins-method-evolve/SKILL.md) | Convert repeated findings into forward-only rubric amendments or capability-map entries. |

## Operating Rule

RUINS remains paper-first. These skills produce markdown research artifacts,
reviews, scorecards, and amendment records. They do not create code, schemas, or
runtime fixtures unless the repo's no-code foundation rule is explicitly lifted.
